Wolfram Research has released Mathematica 5. Over 100 new algorithms for symbolic and numeric computation have been implemented by in-house developers. In addition to updated versions of J/Link and MathLink, Mathematica's standard connections to Java and C/C++, Mathematica 5 comes with the .NET/Link, letting you integrate Mathematica into applications using .NET framework.

SmartDraw.com has released VisualScript XML, a visual scripting tool for XML that lets you program XML statements into graphical symbols, then drag-and-drop the symbols to sketch the flow of a business process. VisualScript translates that sketch into a working XML script. VisualScript XML promotes drag-and-drop diagramming with symbols and intelligent connectors, and is intended to let you build libraries of reusable, preprogrammed symbols to promote your own scripting standards. Other features include the ability to automatically insert user-entered data into XML script output; support multiple XML standards with the same models and templates; and automatically validate XML scripts for proper syntax.

IronTrack SQL generates data comparisons that can pinpoint the cause of SQL performance delays in an application's JDBC performance. IronTrack SQL lets you search for problems resulting from query interactions, or to locate unexplained spikes in database utilization. The tool displays GANTT-style charts of queries as they occur over time, letting you overlay and compare query performance and activity between two or more sets of query-performance data. You can also search for performance problems that result from query interactions, such as when two queries are executed at the same time, and run automated local unit tests to check for a build's variance from an established performance baseline.

LegacyJ has released Version 4.0 of the PERCobol Cobol compiler, designed to allow CICS Cobol applications be recompiled for use in a J2EE environment. PERCobol compiled programs allow access to Java applications through the Cobol CALL statement and with the Callable Program Interface. The tool contains support for more than 15 different dialects of Cobol, including access to most Cobol data sources, and compiles for execution on most platforms and J2EE application servers. PERCobol 4.0 is available with the latest version of the Eclipse IDE.

PrismTech has announced the latest release of its middleware for real-time and embedded ORB and CORBA services, OpenFusion e*ORB 3.0. The new release features an improved, deterministic architecture that is optimized for general-purpose processors, DSPs, and low-power/low-memory embedded microcontrollers. C, C++, Ada, and Java applications can be supported. Additionally, a RT Edition adds support for priorities and scheduling and can be used in any system that requires hard real-time behavior.

Sybase has announced Pocket PowerBuilder, an IDE for mobile and wireless applications. Pocket PowerBuilder features the same 4GL integrated development environment found in PowerBuilder, as well as the Sybase DataWindow for dynamic data access with display formatting and data manipulation capabilities. The tool integrates with SQL Anywhere Studio and includes wizards for easy synchronization, allowing critical data to be communicated back to enterprise systems using wired, public wireless, or Wi-Fi networks. Pocket PowerBuilder synchronizes to a range of databases including IBM, Microsoft, Oracle, and Sybase.
