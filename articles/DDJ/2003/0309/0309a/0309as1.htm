Global Grid Forum, OGSA, and OGSI

The Global Grid Forum (GGF) supports more than 5000 individuals involved with grid technologies in an effort to promote and support the development, deployment, and implementation of grid technologies. GGF produces technical specifications and guidelines that organizations such as the Globus Project can follow in their own grid-development efforts. Globus Toolkit 3, for example, is based on the Open Grid Services Architecture (OGSA; http://gridforum.org/ogsa-wg/) specification developed by GGF to provide interoperability of grid services at the architectural level. An important component of this architecture is the Open Grid Services Infrastructure (OGSI; http://gridforum.org/ogsi-wg/), which specifies how standard web services can be used to support grid computing. Globus Toolkit 3 is a reference implementation of the OGSI 1.0 specification, and as such, enables next-generation computing grids.
