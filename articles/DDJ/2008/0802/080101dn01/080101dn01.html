Dr. Dobb's Journal February 2008

Developer's Notebook

By the DDJ Staff

SftTree/NET 2.0

Softel vdm has released its SftTree/NET 2.0, a combination of tree, listbox, and grid control for .NET. Each cell can contain multiple items—text, images, buttons, checkboxes, radio buttons, progress bars, and more. The control includes support for Visual Studio 2008 and all its supported languages (VB.NET, C#, C++) with Windows Forms. www.windowscontrols.com/treenet

liquid MAPs

mobilePeople has launched its liquid MAPs as a standalone mobile map Java application (like Google maps on mobile) or integrated in mobilePeople's mobile local search solutions. liquid MAPS can also integrate aerial and satellite images within its SVG technology. www.mobilepeople.com

Author-it 5.0

Author-it Software has introduced Author-it 5.0, a content management system (CMS) that includes an interactive authoring memory feature called "Author-it Xtend" that uses fuzzy logic to automate the re-use of content. www.authorit.com

SQL Examiner 2008

TulaSoft has released a pair of database tools. SQL Examiner 2008 compares and synchronizes schemas (structures) between SQL Server databases, letting you merge databases with a few mouse clicks. SQL Data Examiner 2008 automatically compares and synchronizes datasets (database contents) stored in servers of different types, such as SQL Server, Oracle, and MS Access. SQL Data Examiner 2008 also supports error-free automatic cross-platform comparison and synchronization tasks for databases that would otherwise require hours of detailed and tedious work. www.sqlaccessories.com

FirstSQL/J Embedded Mobile Edition

FFE Software has announced availability of FirstSQL/J Embedded Mobile Edition, a version of its standard FirstSQL/J Java database but for small footprints environments. FirstSQL/J Embedded Mobile includes full support for JME (Java Micro Edition) and JSE (Java Standard Edition) in small footprint configurations. It also has components for both JME MIDP (CLDC) and JME Personal Profile/Basis (CDC) and supports SQL92, Intermediate Level. www.firstsql.com

Web.UI for ASP.NET UI

ComponentArt has released Version 2007.2 of its Web.UI for ASP.NET UI controls for web development in .NET. This version adds Editor and SpellCheck controls, as well as the ability for controls to bind directly to Web Services. The Editor is customizable in both look and functionality; the AJAX-enabled SpellCheck control works with the Editor control or independently, supporting custom dictionaries and multiple languages. www.componentart.com

Windows Embedded CE 6.0 R2

Microsoft has released Windows Embedded CE 6.0 R2, which includes a Web-Services-on-Devices API, voice over Internet protocol (VoIP) telephony capabilities, and thin-client technology with auto-detection capabilities. Windows Embedded CE 6.0 R2 lets devices seamlessly connect to computers running Vista and Windows Server 2008, as well as other devices. www.microsoft.com/windowsembedded
