Dr. Dobb's Journal July 2008

Developer's Notebook

By the DDJ Staff

IDEAL

IDEAL Software has released Virtual Print Engine 4.0, a free Community Edition of its PDF Library that runs on Windows, Mac OS X, Linux, and Solaris. The library lets you compute the dimensions of text- and image objects in advance and to position other objects accordingly, so the document layout can be adapted dynamically to the content of objects. The Community Edition ships with .NET and ASP.NET components, an ActiveX, VCL, and DLL/Shared Object, together with demo source codes and extensive help files. The Windows version has an integrated print engine. www.idealsoftware.com

Sun Microsystems

Sun Microsystems has released NetBeans 6.1, which includes rich JavaScript editing features, support for using the Spring web framework, tighter MySQL integration, and an improved way of sharing libraries among dependent projects. Support for Ruby/JRuby has been enhanced with new editor quick fixes, a Ruby platform manager, fast debug support for JRuby, and many other new features and fixes. Sun says that this release also provides improved performance, especially faster startup (up to 40 percent), lower memory consumption, and improved responsiveness while working with large projects. www.netbeans.org

InfomatiX

InfomatiX has released Version 2.0 of its Mobiaccess Mobile Development Framework, a platform-independent mobile SDK. Version 2.0 is platform-independent (Symbian, J2ME, Windows Mobile, WM for Smartphone, BlackBerry), has built-in SSL support, and supports all mobile connection types (BT,WLAN,GPRS/EDGE/UMTS/HSDPA) for data exchange. Mobiaccess 2.0 also handles PIM synchronization, data replication, client-side visualization, and access control on all the platforms. www.mobiaccess.net

TROLLTECH

Trolltech has released Qt 4.4, which provides a common framework for delivering applications and services across both desktops and devices. Qt 4.4 includes an integrated WebKit-based HTML rendering engine, the Phonon multimedia framework, support for Windows CE, and improvements to the QGraphicsView system. Significant is the concurrency framework, an API for developing multithreaded applications and taking advantage of multicore systems. With the Qt WebKit Integration, Qt incorporates WebKit, the open-source browser engine inside Apple's Safari browser, iPhone, and Nokia smartphones. The integration lets developers blend web and native content and functionality and deliver a consistent RIA user experience. www.trolltech.com

BigLever

BigLever Software has released Telelogic DOORS/BigLever Gears Bridge, an integrated solution aimed at helping software development organizations more efficiently manage requirements across an entire portfolio. The DOORS/Gears Bridge solution lets companies leverage the benefits of an end-to-end requirements engineering process within the application development lifecycle, in harmony with other tools and assets across the lifecycle. www.biglever.com

REAL Software

REAL Software is shipping REALbasic 2008 Release 2, a cross-platform development environment for Windows, Mac OS X, and Linux developers. In addition to more than 200 fixes, Release 2 also includes a new class called "Pairs," which lets you define key/value pairs and define linked lists. The release also increases functionality to the introspection feature, which was introduced in the previous release. The new release includes several new capabilities involving remote debugging and bug reporting that simplifies testing and reduces developer overhead. www.realsoftware.com
