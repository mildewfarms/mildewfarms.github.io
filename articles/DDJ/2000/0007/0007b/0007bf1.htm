Figure 1: Modus operandi. 1: rexec requests a host. 2: lmanager requests ldaemon to poll load. 3: ldaemons broadcast loads to lmanagers (to avoid cluttering, only a portion of communication is shown). 4: lmanager selects a host and informs rexec. 5: rexec sends execution request to the selected host. 6: rexecmanger of the host acknowledges. 7: remote sends authentication request to remote ldaemon. 8: ldaemon sends the authentication result. 9: rexec sends the command. 10: remote sets up a channel for standard handles/file descriptors and executes the command.
