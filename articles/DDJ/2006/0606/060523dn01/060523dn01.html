Dr. Dobb's Journal June 2006

Developer's Notebook

McObject

McObject has released eXtremeDB 3.0, an upgrade to its in-memory embedded database for intelligent devices. New eXtremeDB 3.0 features include a remote access interface, support for nonvolatile RAM, and task scheduling that uses a new synchronization mechanism to minimize expensive context switches. These features enable a process to reconnect to the NVRAM-hosted database, initiate any needed cleanup, and resume normal operation after the system reboots. www.mcobject.com

Visual Numerics

Visual Numerics has released Version 6.0 of its IMSL C Numerical Library, a comprehensive set of thread-safe functions for mathematical and statistical applications. Version 6.0 includes a new Neural Network Engine, Dense Linear Programming Optimizer, Differential Equations Package, random number generator technique using the Mersenne Twister algorithm, and a new LAPACK (Linear Algebra PACKage) functionality. www.visualnumerics.com

UModel 2006

Altova has released Altova UModel 2006, a UML-based modeling tool. In addition to supporting UML diagram types and Java code, Altova UModel 2006 provides support for the UML 2.1 and XMI 2.1 specifications, sequence diagrams, and C# code generation, reverse engineering, and round-trip engineering. www.altova.com

Superior SQL Builder

Red Earth Technologies has announced its Superior SQL Builder Standard Edition, an edition of Superior SQL Builder that contains most of the features of Superior SQL Builder Professional Edition, but at a lower price. Like the Professional Edition, Superior SQL Builder Standard Edition helps you develop complex SQL scripts via a user-friendly interface and script templates. Superior SQL Builder supports Microsoft SQL Server, Microsoft Access, MySQL, and Oracle. www.superiorsql.com

Almeza Research Multiset

Almeza Research has released Multiset, an automatic program installer for Windows that starts installation procedures—taking care of all necessary buttons in the process. This installation robot specifies the path for the installation and automatically configures all attending settings. All registration codes and license keys are also entered in a self-acting mode. Multiset's benefits include full program installation optimization, no need to write code/scripts, and a multilingual interface. www.almeza.com

Protector for PHP

ByteRun Software has released Protector for PHP, a bytecode compiler and encoder that lets you quickly and easily protect source of PHP scripts before distribution. Protector for PHP is the standalone Windows application without special requirements, protects any number of files at once, and has special features like script expiration or domain lock. The ability to create PHP installers greatly simplifies script deployment. Produced installers can also be protected by password. www.byterun.com

Parasoft SOAtest 4.5

Parasoft has released Parasoft SOAtest 4.5, its automated software test, analysis, and development governance solution for web services and Service Oriented Architecture (SOA) implementations. Release 4.5 supports infrastructure and governance in the development of web services. These include enhanced SOA development governance, test management support, and new automated testing of BPEL (Business Process Execution Language) and BPEL-based processes. www.parasoft.com

Jungo WinDriver USB/PCI 8.0

Jungo Software Technologies has released Version 8 of its WinDriver USB/PCI development tool for high-performance, cross-operating system user-mode USB/PCI device drivers that support Windows, Windows CE, Linux, Solaris, and VxWorks. WinDriver USB/PCI 8.0 supports Windows XP 64-bit environments and Windows Server 2003 64-bit environments (x64) on the Intel EM64T and AMD64 architectures. WinDriver 8 now also supports automatic driver code generation in C# and Visual Basic .NET, as well as driver projects for the latest Visual Studio 2005 development environment. For Linux, WinDriver 8 supports the Linux kernel 2.6.14, and completes the support for 64-bit environments. www.jungo.com
