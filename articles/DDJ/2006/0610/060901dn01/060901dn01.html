Dr. Dobb's Journal October 2006

Developer's Notebook

Borland

Borland has gone back to its roots, with new editions of its renowned Turbo language tools. The initial release includes Turbo C++, Turbo C#, Turbo Delphi, and Turbo Delphi for .NET, each available in a free Turbo Explorer and commercial Turbo Professional edition. Each package includes more than 200 prebuilt drag-and-drop components. http://www.borland.com

Wingware

Wingware has released Version 2.1 of Wing IDE for Python. Wing IDE provides debugging, editing, code intelligence, and search capabilities that reduce development and debugging time, cut down on coding errors, and make it easier to understand and navigate Python code. Highlights of the Wing IDE 2.1 release include: Visual Studio, VI/Vim and Brief key bindings, Subversion and Perforce support, improved performance on Windows, named bookmarks, breakpoint manager and call stack as list, and support for Intel Macs. http://wingware.com

Klocwork

Klocwork is shipping Version 7.1 of its static-analysis product suite. Klocwork K7.1 includes new checkers for C, C++, and Java source-code analysis, in addition to enhanced capabilities that make it easier for K7.1 to be integrated into complex development environments. Klocwork's static analysis suite enables cross-functional development teams to quickly identify software defects and vulnerabilities. The new version integrates with complex build systems and automatically recognizes any changes to a customer's build. Klocwork also added 44 new vulnerability checkers across Java, C, and C++, and features ARM Compiler and Java 1.5 support. http://www.klocwork.com

Elementool

Elementool, a provider of bug, defect, time-tracking, and help-desk tools for web-based project-management tools, has announced the release of a bug-tracking add-in tool for Visual Studio.NET 2005. It is available free to Elementool's project-management subscribers. This release of the Elementool add-in uses Elementool's own "Bug Life Cycle Model" to track and resolve issues during development. http://www.elementool.com

Symbian Limited

Symbian Limited has announced Symbian OS 9.3, the most recent version of its smartphone operating system. Symbian OS 9.3 supports development tools such as the Eclipse/CDT IDE framework and Nokia's Carbide.c++ Development Tools for Symbian OS 9.3 phones, and provides configuration tools to create and customize Symbian OS variants, the Symbian Verification Suite to support compatibility and phone integration for creating Symbian OS variants. A reference design for Symbian OS 9.3 with Freescale and Nokia S60 is included. New hardware support includes native support for WiFi and USB 2.0 on-the-go. http://www.symbian.com

EmbeddedPlus Engineering

EmbeddedPlus Engineering has released its SysML and Simulation Toolkits, add-ins for the IBM Rational Software Development Platform. The SysML Toolkit lets teams develop and share models in Eclipse-based modeling environments. The Simulation Toolkit allows for SysML and UML models to be simulated and executed. SysML, which is based on UML, provides support for specifying, analyzing, designing, and verifying complex systems that may include hardware, software, information, personnel, procedures, and facilities. Both toolkits have received Ready for IBM Rational Software validation. http://www.embeddedplus.com

TechExcel

TechExcel has released DevPlan, a project-planning tool designed to manage application development. DevPlan combines with TechExcel's DevTrack to provide an integrated project-planning and management solution. In addition to familiar planning features such as a Gantt chart view, resource management, and baseline comparisons, DevPlan provides features such as integrated knowledge management, issue-tracking, and team-based planning and calendaring capabilities. http://www.techexcel.com
