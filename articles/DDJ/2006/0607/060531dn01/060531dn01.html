Dr. Dobb's Journal July 2006

Developer's Notebook

by DDJ Staff

Flux

Flux has released Version 7.1 of its Java job scheduler, workflow engine, and business process management (BPM) engine. Flux 7.1 has a web-based Operations Console that graphically monitors jobs and workflows in near-real-time. Failed jobs and workflows can also be reviewed, recovered, and returned to service. The Ajax-enabled Flux 7.1 also adds agents that let system processes be executed on remote computers to increase overall system throughput. www.fluxcorp.com

RKD Software

RKD Software has released BarCode ASP.NET Web Control 1.5, an ASP.NET component that lets you add barcodes to ASP.NET web applications. The component integrates into Visual Studio, and you can customize barcode properties in its Properties window. It is written in C# and is 100-percent managed code. BarCode ASP.NET Web Control supports all popular barcode symbologies. www.barcodetools.com

Actional

Sonic Software has made available Release 6 of Actional, its management platform for service-oriented architectures (SOA). Major functionality in Actional 6.0 includes automated runtime governance, which provides auto-discovery and autoenforcement of policies for services and processes, and "Trust Zones" security to prevent unauthorized or noncompliant usage of services. www.sonicsoftware.com

The JVM Tuning and Benchmarking Service

Sun Microsystems has announced a tuning and benchmarking service for optimizing performance of the Java SE platform for embedded deployments. The JVM Tuning and Benchmarking Service focuses on adjusting the numerous tunable components in the standard Java VM, including stack memory, JIT compiler settings, selecting the proper garbage collector, and other settings. java.sun.com/j2se/embedded

Compuware TestPartner

Compuware has introduced Compuware TestPartner 5.4, which offers a tight functional testing integration with Visual Studio 2005 Team System and support for the .NET Framework 2.0. TestPartner is an automated testing tool for functional testing of complex applications. 5.4 features include an ActiveData Wizard for automatic data parameterization, an Import/Export Wizard, expanded support for Java and SAP technologies, and test results summary. www.compuware.com

Dynamic PDF Generator 4.0 for .NET

ceTe Software has released DynamicPDF Generator 4.0 for .NET, a class library that lets you programmatically create PDF documents from applications, and DynamicPDF Merger 4.0 for .NET, which lets you programmatically merge, split, stamp, form-fill, and personalize PDF documents. www.cete.com

CryptoStick Software Suite

Research Triangle Software has announced that its CryptoStick Software Suite can now be loaded onto iPods and other portable devices for use with Windows-based PCs. The software is carried on the device—not the hard drive of the Windows-based computer—so all encryption and other security functions are local to the portable devices. www.rtsz.com

Project Management Methodology

Method123's Project Management Methodology is a suite of project management templates, project forms, and project documents used to initiate, plan, execute, and close projects. It includes more than 50 Word and Excel project templates, comprising more than 300 pages of hints, real-life examples, charts, and tables. www.method123.com
