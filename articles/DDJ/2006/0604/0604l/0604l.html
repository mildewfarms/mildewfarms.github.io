The Eclipse Foundation has released the Eclipse Web Tools Platform (WTP) 1.0. Eclipse WTP 1.0 is an extensible, Standards-based tool platform for developing J2EE and web applications. WTP has already been adopted by BEA, Borland, IBM, JBoss, and ObjectWeb. This full version release solidifies the platform API for third-party extension, and accompanying documentation improvements make WTP ready for the next-level of vendor adoption. This release focuses on two of the three major goals outlined in the WTP roadmap—defining the WTP API and keeping current with the underlying Eclipse platform. Future releases will focus more on the third roadmap goal of currency with Standards like Java 5.0 and WSDL 2.0.

Helixoft has released VBdocman .NET 2.2, an add-in for Visual Studio .NET 2002/2003/2005. This program lets you generate technical documentation from Visual Basic .NET and ASP .NET projects with a few mouse clicks. VBdocman parses the source-code files and automatically creates the table of contents, index, topics, cross-reference, IntelliSense information, and context-sensitive help. You can also use VBdocman to add C#, XML, or JavaDoc comments to the source code via the WYSIWYG comment editor, which enables you to insert tables, lists, pictures, links, and other formatting elements directly to the code. VBdocman compiles documentation into a fully customizable HTML Help, Microsoft Help 2, HTML, RTF document, or XML file.

TulaSoft has released SQL Examiner 1.2, a database synchronization tool for automatically moving changes from development databases to production servers. Its engine compares objects of different hierarchy and builds a script for a fast and accurate synchronization. Stripped of the service code, the synchronization script is easy to modify for your specific needs. SQL Examiner 1.2 passes over noncritical tasks to provide higher synchronization speed while maintaining accuracy. SQL Examiner 1.2 compares and synchronizes different SQL versions; supports SQL 2005 Server, including new data types, indexes, schemas, and triggers; supports synchronization of privileges and user roles, and synchronization of a back up copy (before extracting), detached database and snapshots; and supports partial synchronization and exclusion of dependents.

Intel has released beta software development tools and resources designed to help Apple developers take advantage of Intel's Core-Duo processor technologies on the Intel-based Macintosh platforms. Special versions of the Intel Fortran and C++ compilers, and Math Kernel Library and Integrated Performance Primitives are available. Intel will also provide other resources to assist with optimization, dual-core threading, and information migration. These development tools are integrated into Apple's Xcode development environment.

Aonix has released ObjectAda Windows for the PowerPC/VxWorks development environment. Windows-hosted ObjectAda 8.2 improves the underlying Aonix Ada 95 compiler, reducing compile time for the PowerPC/VxWorks platform. ObjectAda 8.2 includes the capability to attach the symbolic debugger to a running Ada application, which aids in resolving programming errors discovered after the test and debug phase is completed. Integration with the VxWorks environment is seamless thanks to a specifically developed VxWorks binding that enables Ada and VxWorks tasks to be combined in the same application.

SpeeDEV has released Version 4.5 of its SpeeDEV application development lifecycle tool. New features in SpeeDEV 4.5 include custom entity definition, a SOAP-based API, and custom cross-project reporting. SpeeDEV 4.5 is designed to foster better collaboration among disparate business and technical groups. Additionally, Release 4.5 improves performance and enhances many of the existing features, such as linked custom fields, advanced search, configurable homepage, and more.

IBM has announced IBM Rational Portfolio Manager 6.2, a project and portfolio management solution for incorporating software development best practices into executable project plans. The software gives you an analytical dashboard that helps automate decision and approval cycles for teams while spotting trends and overlaps in projects. Using IBM Rational Portfolio Manager 6.2, you can prioritize projects, identify overlaps, and manage the execution of specific software development tasks. The software helps you make informed decisions about which projects to pursue, based on their potential impact to the business and availability of resources. Once a project is selected, IBM Rational Portfolio Manager helps ensure the project is effectively managed to reduce unnecessary costs and completed in a timely manner. The package includes a Standards-based API SDK, the IBM Rational Method Composer development process solution, and graphical workflow capabilities.
