C Preprocessing With Tcl

By Jonathan S. Arney

Dr. Dobb's Journal August 1998 

(a)
set a "abcdefg"
puts stdout $a
(b)
proc print_a_string {a} {
	puts stdout $a
}
print_a_string "abcdefg"

Example 1: (a) Sets the variable a to "abcdefg," and prints its value to the standard output; (b) a Tcl function.
