Template Processing Classes for Python

By Brad Howes

Dr. Dobb's Journal February 1998 

>>> from BoilerPlate import String
>>> z = '''a
#if b#
b
#else#
c
#endif#'''
>>> print String( z, lineMode = 0, b = 1 )
a
b
>>> print String( z, lineMode = 1, b = 1 )
ab
>>> print String( z, lineMode = 1, b = 0 )
ac
