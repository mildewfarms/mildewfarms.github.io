Template Processing Classes for Python

By Brad Howes

Dr. Dobb's Journal February 1998 

>>> from BoilerPlate import String
>>> z = 'A#if a == 1#B#elif a == 2#C#else#D#endif#E'
>>> print String( z, a = 1 )
ABE
>>> print String( z, a = 2 )
ACE
>>> print String( z, a = 99 )
ADE
