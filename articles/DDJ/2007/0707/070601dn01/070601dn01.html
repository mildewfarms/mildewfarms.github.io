Dr. Dobb's Journal July 2007

Developer's Notebook

By the DDJ Staff

Paragon NTFS

The Paragon Software Group has released Paragon NTFS for Linux 6.0, which provides rapid and transparent read/write access to NTFS volumes under Linux. Among the features is support for: 64-bit CPU architectures, Windows Vista NTFS filesystem, and the latest Linux kernel. Paragon's Universal File System Driver SDK provides tools and sample code for building support for unsupported filesystems into applications. www.paragon-software.com

010 Editor 2.1

SweetScape Software has released 010 Editor 2.1, an editor for Windows for modifying binary files, drives, or processes. Using 010 Editor's Binary Templates technology, binary data can be viewed/edited. 010 Editor lets Binary Templates be shared among users, and features an online repository of Binary Templates developed by users. The repository contains templates for file formats such as AVI, BMP, GIF, PAL, RAR, TIF, WAV, ZIP, and more. www.sweetscape.com

Carbide.c++ Version 1.2

Nokia has unveiled Version 1.2 of its Eclipse-based Carbide.c++ developer tools. Among the enhancements are a new Build system with new wizards to detect more types of kits and import more projects. New optimization tools for both power and memory are also included. The system also now supports Eclipse 3.2.1, and the CDT 4.0 M5 release. A new GUI Crash Debugger enables the debug context of crashes. www.forum.nokia.com/carbide

PeakStream Workstation for Microsoft Windows Edition

PeakStream has announced PeakStream Workstation for Microsoft Windows Edition that lets you program high-performance processors such as multicore CPUs and graphics processor units (GPUs) directly on your desktop. PeakStream Workstation for Microsoft Windows beta combines a set of extensive Math libraries in C/C++ with an optimizing runtime and a set of Visual Studio 2005 tool extensions for rapid application development. www.peakstreaminc.com/microsoft

NetAdvantage for WPF 2007

Infragistics announced NetAdvantage for WPF 2007 Volume 1, a toolset that extends the capabilities of the Windows Presentation Foundation (WPF) by providing high-performance, editable DataGrid style controls. Using the capabilities provided by the WPF platform such as vector-based graphics, subtle animation, and high stylization capabilities, NetAdvantage for WPF provides tools for creating user interfaces. www.infragistics.com/wpf

Code Collaborator 2.1

Smart Bear Software has released Code Collaborator 2.1, which facilitates peer code reviews. Features unique to Version 2.1 include: externalized defects that let you return to errors to fix later on, keyboard shortcuts, Oracle support, SSL support, Eclipse support for ClearCase and Subversion, and CMVC integration for IBM's internal version-control tool, among others. www.smartbear.com
