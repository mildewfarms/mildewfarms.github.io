Dr. Dobb's Journal January 2007

Developer's Notebook

By the DDJ Staff

Backbase

Backbase has released its Backbase AJAX Struts Edition, which combines Struts with AJAX for building rich Internet applications by adding rich user interfaces, without having to rebuild the entire application. The Backbase AJAX Struts Edition includes both AJAX technology and Struts-specific functionality in the form of Struts Taglibs. The Backbase AJAX Struts Edition also includes an Eclipse-plugin and can be deployed on BEA WebLogic, IBM WebSphere, Oracle 10g, JBoss, and Tomcat. www.backbase.com

Carbide.ui Theme Edition

Nokia has announced is Carbide.ui Theme Edition, a PC-based tool for building customized user interfaces for mobile devices. The Carbide.ui Theme Edition provides WYSIWYG and drag-and-drop functionality, plus seamless interplay with leading graphics creation tools for customization of smartphone look-and-feel. The tool includes nearly 1000 customizable drag-and-drop elements to create, edit, and view UI theme components, including wallpaper and backgrounds, application icons, pop-up windows, and color schemes. Once a UI theme has been created, the tool automatically packages it into an installation file and prepares it for Open Mobile Alliance (OMA) Digital Rights Management (DRM) protection, if required, after which it will be ready for deployment to devices. www.forum.nokia.com/carbide

Intel Cluster Toolkit 2.0

Intel has announced Version 2.0 of its cluster toolkit for Linux. The Intel Cluster Toolkit 2.0 for Linux is designed for building scalable parallel message-passing interface (MPI) apps that execute across multiple network architectures, collect trace information, and analyze runtime behavior on clusters. It runs on Red Hat Enterprise Linux 3/4 and SUSE Linux Enterprise Server 9, and compatible with Intel's C/C++ and Fortan compilers for Linux. www.intel.com/software/products

Mobile JUnit 1.0

Sony Ericsson has released Mobile JUnit 1.0, a framework for the automated testing of mobile applications targeting Java ME Connected-Limited-Device-Configuration (CLDC) phones. The Sony Ericsson Mobile JUnit enables fully automated regression testing of CLDC applications both in the emulator and on the real phone. process for finding and debugging code. Mobile JUnit and a paper can be downloaded for free from the Java Docs & Tools section of the Sony Ericsson Developer World. www.sonyericsson.com/developer/java

Infragistics NetAdvantage

Infragistics has released NetAdvantage for .NET 2006 Volume 3, which lets you create ASP.NET and Windows Forms applications with the look-and-feel of Microsoft Office 2007, using Microsoft Visual Studio 2005/2003/2002. The tool includes a number of new features and functionalities, including the ability to use built-in AJAX behaviors seamlessly alongside Atlas. www.infragistics.com/netadvantage/

IT Governance

IT Governance has released its IT Governance Framework Toolkit which provides the tools for understanding, organizing, adopting, and monitoring IT governance practice. Its 98 separate documents include templates, guidelines, checklists, questionnaires, slide presentations, assessments, and planning tools. The Tookit is based upon the Calder-Moir IT Governance Framework, which provides an end-to-end method for organizing IT Governance throughout organizations. www.itgovernance.co.uk
