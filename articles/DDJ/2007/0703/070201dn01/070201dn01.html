Dr. Dobb's Journal March 2007

Developer's Notebook

By the DDJ Staff

Java Web Components

JSCAPE and SoftAspects have released WebGalileo Faces 3.0, a suite of Ajax-enabled Java Server Faces (JSF) components for developing web-based Rich Internet Applications. WebGalileo Faces 3.0 includes components for creating graphs, flow charts, and maps using Google Maps. WebGalileo Faces has built-in support for Sun Java Studio Creator, IBM Rational Application Developer, Oracle JDeveloper, and Eclipse IDE. www.javawebcomponents.com

Hansoft

Hansoft has announced that its Hansoft 4.0 Project Management and Collaboration tool now supports Agile development methodologies. With the 4.0 Agile release of Hansoft, you can in one tool use Agile and task-scheduling projects in parallel. You can also have Agile cells of subteams within a development project. With Hansoft used in Agile projects, workload and portfolio management, history, ghost resources, client and outsourcing partner integration, and document management are available. www.hansoft.se

Syncfusion

Syncfusion has announced the release of Essential Studio 4.4, which includes support for the new Office 2007 look-and-feel. In addition, 4.4 includes ATLAS support for all of Syncfusion ASP.NET controls and a revamped look-and-feel for its Essential Grid. www.syncfusion.com

Polar Rose

Polar Rose has released tools for combining automatic face recognition with 3D modeling for visual search applications. Available as a free web browser plug-in and through royalty-free APIs, you can create applications that let you automatically search for and recognize faces on a consistent basis similar to the way people do—by using the visual cues of the photo as opposed to nearby text analysis. The technology creates a 3D model from a single 2D image of a face. www.polarrose.com

CrypKey

CrypKey has released CrypKey SDK 7. The CrypKey SDK supports true client/server network licensing, so that a software license can be shared with any computer on a network without needing a shared drive. Other features of Release 7 include: applications that can act as either license servers or license clients; automatic restart of servers upon reboot; support for an unlimited number of redundant servers; and the ability to specify domains, computers, and users that can or cannot use a particular license. www.crypkey.com

Recursion Software

Recursion Software released Cinergi 2.1, a multilanguage application integration platform (ESB) that supports service-oriented architectures (SOA). Cinergi 2.1's runtime environment for C++ provides 64-bit support for AIX 5.3 and Linux (Red Hat). Developers using Cinergi's native Windows-like Architect Console can fully leverage 64-bit CPUs for increased performance. Cinergi 2.1 also has a failsafe feature that protects against server failure once the runtime application starts. Cinergi clients and servers communicate regardless of the ability to communicate with the license server. www.recursionsw.com
