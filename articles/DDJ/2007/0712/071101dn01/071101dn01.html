Dr. Dobb's Journal December 2007

Developer's Notebook

By the DDJ Staff

Codign Testing Tools

Codign Software has upgraded its CoViewDeveloper and CoViewManager testing tools. Software teams can determine what makes their code testable by selecting up to nine metrics, determining their threshold, and setting their severity level. CoView tells you what methods/classes/packages exceed the thresholds in real time via syntax-like decorators. www.codign.com

ShipRush

Z-Firm has released Version 5 of its ShipRush SDK, for building applications that involve parcel shipping. With the SDK, you can embed parcel shipping into nearly any .NET, ASP.NET, VB6, Visual C++, Delphi, FoxPro, or other Windows or web application. The SDK includes visual and non-visual components and a Web service. ShipRush calculates shipping rates and prints shipping labels for UPS, FedEx, DHL, and USPS shipments. www.zfirm.com/shiprush_sdk

TrueUpdate

Indigo Rose Software has announced TrueUpdate 3.0, a tool that lets you integrate a fully automatic "check for update" feature into applications. Once a required patch has been identified, the software's client automatically retrieves and applies the update using standard protocols. TrueUpdate works with any standard web server. www.indigorose.com

Aladdin

Aladdin Knowledge Systems has released Version 2.5 of its HASP Software Rights Management software. The .NET Envelope for Framework 3.0 lets HASP automatically protect software to the Method-level and assign licenses to chosen Methods on Frameworks 2.0 and 3.0. Method-level encryption is the highest level of security available in a DRM product. www.aladdin.com

AdaCore

AdaCore's GNAT Pro development environment is available for .NET Framework, making it the first commercial Ada tool to support the .NET through unmanaged (Windows) and managed .NET code. The product includes an Ada compiler (which supports the new Ada 2005 standard,) toolset, and libraries and bindings. www.adacore.com
