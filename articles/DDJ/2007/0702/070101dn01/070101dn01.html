Dr. Dobb's Journal February 2007

Developer's Notebook

By the DDJ Staff

Ajaxium

Ajaxium 2.0 from Ajaxium is a component that acts as a universal AJAX container for default and custom ASP.NET controls and pages. Ajaxium-powered pages become AJAX-enabled, but preserve their normal lifecycle. In addition to empowering all controls on separate ASP.NET pages, Ajaxium handles all types of hyperlinks, enabling smooth navigation from one AJAX-enabled page to another and letting you AJAX-ify entire web sites. All pages remain compatible with all browsers and search engines. www.ajaxium.com

Advanced Installer

Caphyon has released Advanced Installer Professional 4.5, a Windows Installer authoring tool that lets you build MSI packages. Advanced Installer runs on Windows 2000/XP, and the install packages it creates run on Microsoft Windows 9x/ME/NT/XP. Advanced Installer project files are saved as XML files for integration with source control systems. www.advancedinstaller.com

Oracle Developer Depot

Oracle has announced its Oracle Developer Depot, a free tool for finding, configuring, and provisioning Java applications for prototyping. Oracle Developer Depot leverages Web 2.0 technologies such as AJAX, the Spring 2.0 Framework, and RSS to facilitate code reuse and simplify development. Once deployed, Oracle Developer Depot can establish RSS-based communication between Oracle Application Server 10g and a remote repository for application and code archives such as the Oracle Technology Network. oracle.com/technology/goto/oracle-developer-depot

Web.UI.2006.2

ComponentArt has announced its Web.UI 2006.2 for ASP.NET AJAX, a suite of controls for Microsoft's ASP.NET AJAX development framework (formerly "ATLAS"). Web.UI for ASP.NET AJAX offers deep integration into the ASP.NET AJAX framework: All controls inherit from the Sys.UI.Control client-side base class, implement type descriptors, and expose extensive client-side object models built specifically for the AJAX Library type system. atlas.componentart.com

JNBridgePro 3.1

JNBridge has released Version 3.1 of JNBridgePro, its Java and .NET interoperability tool. JNBridgePro 3.1 lets you embed Java GUI controls directly in .NET applications, or .NET GUI widgets in Java applications. With this version, you can embed Java widgets written in AWT or Swing directly inside WinForm applications, or embed .NET WinForms controls in to Java AWT, SWT, or Swing applications. JNBridgePro 3.1 works with Visual Studio 2005, .NET Framework SDK 3.0, and JDK 1.3.1 or later (including J2SE 5.0 SDK). www.jnbridge.com

MyEclipse

Genuitec has made available MyEclipse Enterprise Workbench 5.1, which bundles Java 5 JRE, Eclipse 3.2.1, and MyEclipse 5.1 together in a single download. The workbench provides Eclipse 3.2.1 compatibility, and supports custom user forms, custom controls, and improved usability. An enhanced Hibernate HQL editor supports multiproject configuration and MyEclipse is now i18N enabled. www.myeclipseide.com
