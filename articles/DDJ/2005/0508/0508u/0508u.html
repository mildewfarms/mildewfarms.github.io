Code Canvas Technologies today announced the release of RapidJ, a Java code generator that creates fully functional and customizable J2EE web applications. RapidJ generates tiered J2EE web applications that utilize technologies such as Struts, Java Server Pages (JSP), and Java Database Connectivity (JDBC). Standards compliant, the software also provides specific support for many of the common application and database servers. Integrated with RapidJ are the Resin application server and HSQLDB database. RapidJ requires Windows or a UNIX-based operating system, Java Standard Edition Version 1.4 or better, and Apache Ant.

db4objects Inc. has announced the release of Version 4.5 of db4o, its open-source object database for Java and .NET. db4o employs object-oriented replication, which is simpler than conventional replication out of relational database tables. As a client-side, embeddable database, db4o is suited for mobile devices or products with Java and .NET software. Additional improvements in db4o 4.5 include a pluggable reflector and a new generic reflector that help you build more diverse architectures, along with a new object-oriented ObjectManager for easy browsing of database files.

Electric Cloud has released Electric Cloud 2.2, which improves build throughput and enhances support for centralized build clusters. At the core of Electric Cloud 2.2 is Electric Make, which is compatible with existing Make-based build infrastructures so it can be rapidly deployed with minimal change to existing tools or build environments. Electric Cloud 2.2 offers a command-line interface to the cluster resources, in addition to its web-based User Interface. In Electric Cloud 2.2, cluster sharing has been improved to more efficiently allocate agents to builds.

PowerProgrammer has released Version 3.0 of its Web Update Wizard which adds automatic update functionality to your applications with a single line of code. The Web Update Wizard runs on all versions of Windows and works with Visual Basic, C++, Delphi, or any other development environment capable of making Windows DLL calls or launching a Windows process. Every time a user runs your software, the Web Update Wizard checks for updates, then downloads and installs them as required.

The Coding Workshop has announced the launch of Polyphonic Wizard Version 4. The Polyphonic Wizard software runs on your PC and lets you convert standard music files (midi and mp3) into polyphonic ring tones without needing any musical knowledge. The software includes all the necessary drivers and connections to convert a midi file, and subsequently download a ring tone to a phone without the need for cables. All you need is a basic PC and an Internet connection. Supported phones include most polyphonic handsets from Alcatel, Kyocera, LG, Motorola, Nokia, NEC, Panasonic, Samsung, Sony Ericsson, Sendo, Sharp, and Siemens. Support for more handsets is added every month.

Koders has unveiled Koders.com, a search engine for identifying and accessing open- source code on the Internet. The company currently indexes over 190 million lines of code from more than 28,000 projects in hundreds of open-source repositories that you can search by keyword, language, and license or perform advanced queries using enhanced syntax. Search results are then displayed in a developer-friendly format that makes it easy to understand the code in its original working context.

DataDirect Technologies has announced its DataDirect XQuery, an embeddable XQuery implementation for XML applications that need to process both XML and relational data sources. DataDirect XQuery lets you query XML, relational databases, or a combination of the two, integrating the results for XML-based data exchange, XML-driven web sites, and other applications that require or leverage the power of XML. With DataDirect XQuery, you can programmatically invoke and process XQuery expressions against any major relational database including Oracle, Microsoft SQL Server, and IBM DB2, directly from within your Java applications.

Cogent Real-Time Systems announced the release of a free API for its Cascade DataHub that gives you access to dynamic data for monitoring, conditioning, and manipulation, and opens possibilities for connections to legacy programs. Available in C++, Java, and C#, the DataHub API uses a straightforward, object-oriented syntax to interact with the streaming data collected and distributed by the Cascade DataHub. The Cascade DataHub lets users share real-time data among any number of applications using standard protocols and a scalable technology. It creates dynamic data links among Windows or Linux computers, over the Internet, or on local area networks.
