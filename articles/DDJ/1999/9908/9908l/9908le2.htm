(a)
try{ 
  operationA();
  operationB();    
  operationC(); }
catch(Exception e) { 
    catchOperationA();  }

(b)
try{
  operationA();
  operationB();
  operationC(); }
catch(NullPointerException e) {
     catchOperationA(); }
catch (Throwable e) {
     catchOperationB(); }
         

(c)
try{ 
  operationA();
  operationB();    
  operationC(); }
catch(Exception e) { 
    catchOperationA();  }
finally { cleanupOperation(); }

Example 2: Optional features of the try/catch mechanism. (a) Control flow transfer; (b) multiple catch blocks; (c) finally clause.
