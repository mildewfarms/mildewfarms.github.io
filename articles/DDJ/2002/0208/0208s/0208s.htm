Anysoft has released Digital Cortex, a software sequencing system that creates an open windows environment where applications become accessible and externally programmable to transcend standards and programming languages. It noninvasively exposes windows (console applications, terminal emulators, Java, client-server, desktop, and web based) as programmable objects at run time, in real time. The Digital Cortex system works with any programming language that supports COM/ActiveX to leverage existing toolsets to create programmatic access to applications without requiring source code, extend applications to new delivery platforms (Web and wireless), and reuse functionality and data from existing applications to create new applications and web services.

Colasoft.com has announced the release of Colasoft Application Protocol Sniffer & Analyzer (Capsa) 2.1, a tool for TCP/IP network monitoring and analyzing. The program lets you trace TCP connection states and monitor IP packets over the local network host. It supports and decodes protocols such as POP3, SMTP, HTTP, TELNET, FTP, and more. Capsa also gives you control over which packets you want to reject with customizable filtering options. It captures data in real time from a live network connection, and lets you analyze data from a capture file. Packet viewing options let you customize how the captured packets are viewed, and filter any duplicates as well as copy-captured data. Capsa is available for all Windows platforms.

CodeCharge, developed by YesSoftware, is a RAD tool that uses a code-generation engine to automatically generate web applications in many server-side languages including C# (ASP.NET), VBScript (ASP), ColdFusion, Java (JSPor Servlets), PHP, and Perl that support databases such as Oracle, Sybase, MySQL, SQL Server, and Access. CodeCharge lets users assemble large-scale web applications rapidly and is specifically aimed at creating data-driven web applications. It includes a full-featured IDE with a visual HTML design interface and HTML code editors, drag-and-drop components, and a Power Wizard capable of quickly converting database tables into working web applications.

Sakrament has released the Sakrament Text-to-Speech Engine 2.0. This speech synthesis module is developed on a special acoustic base with the use of sound processing algorithms that allow the creation of near natural sounding, high-quality synthesized speech. Version 2.0 is compliant with the Text-to-Speech Engines Industrial Standard Microsoft Speech API (SAPI) 5.1; is able to work in telephone lines using TAPI 3; can be used with any application supporting SAPI 5.1 protocol; includes multilingual and multivoice support; and supports a full set of SAPI Speech Tags, and additional tags. The Sakrament Text-to-Speech Engine 2.0 can be used in speech portals, online information services, electronic devices, software and hardware products, telecommunications, and more.

SHUNRA\Forecaster is a tool to integrate network emulation with load testing. For its WAN emulation component, Forecaster uses technology from SHUNRA\Storm, which supports a variety of network connections ranging from slow modems to OC-12 speeds and from low-latency links to high-latency satellite links. The combined emulation of the network and user load lets Forecaster graph baseline performance information, forecast the impact of network upgrades on end-user response times and evaluate needed SLAs.
