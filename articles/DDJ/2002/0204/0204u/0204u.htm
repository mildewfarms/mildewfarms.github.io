Dr. Dobb's Journal April 2002

Borland Software has released Borland Platform for J2EE, which consists of Enterprise Studio for Java, JBuilder 6 in Enterprise, and Enterprise. The Borland Enterprise Studio for Java features JBuilder; Rational Rose, a modeling environment; and best practices methodology through Rational Unified Process. The Studio is also available with Macromedia Dreamweaver UltraDev 4. Borland JBuilder 6 features a new way to create EJB components through point-and-click visual development. It also integrates and supports features of extreme programming such as refactoring, unit testing, and automated Javadoc creation. The Borland Enterprise Server is now available in three editions: Web, VisiBroker, and AppServer edition.

LIFT from UsableNet is an add-on product for developing accessible and usable web sites within Macromedia Dreamweaver 4 and Dreamweaver UltraDev 4. This software includes Accessibility Suite, a solution including fix wizards, monitoring, reporting, and new usability options. Some new features include: customizable guidelines, the ability to automate the fixing process, elimination of manual evaluation of HTML, and the ability to review the development and design in real time.

Sonic Software has announced SonicXQ, a tool for web services. It provides XML processing support and integration on a secure platform. SonicXQ is fully standards based, provides any-to-any application connectivity, and provides fully secure HTTP/S and SOAP support as well as WSDL generation for UDDI registries. It also has a message-based architecture that routes and transforms critical business information between services according to configurable itineraries, and supports JCA adapters to transport common application connectivity to web services through SOAP.

X-Hive has released X-Hive/DB 2.0, a native XML database. Features include document versioning, which is the ability to store multiple versions of the same document without sacrificing the database's speed and performance; binary object (BLOB) support, which allows binary large objects (such as movie files, images, or Microsoft Office files) to be stored in X-Hive/DB; and full-text searching, which is accomplished via integration with Lucene, the open-source full-text index engine from Apache. Prior support for node indexes (Context Conditioned Indexing) and element name indexes has been expanded to include library indexes, ID attributes, and full-text indexes.

Waterloo Maple has released Maplets, an add-on package for Maple 7 that lets you create and customize your own GUI for Maple. It can be used to create custom Maple calculators, interfaces to Maple packages, queries, and messages. With Maplets, users can present Maple applications through slider bars, buttons, text areas, pull-down menus, and other GUI elements, to other users who do not use Maple. Maplets invoke GUI elements for Sun's Java Swing package, but no Java programming knowledge is necessary to develop a Maplet interface.

Metrowerks has released the CodeWarrior IDE for the MIPS-based RISC embedded systems architecture. New in Version 4.0 of CodeWarrior IDE is support for the MIPS32 Instruction Set Architecture (ISA). In addition, it provides you with usability features unique to MIPS tools including drag-and-drop editing and file management. The tools also support IDT 32300 RISCore, LSI EZ4103, NEC Vr5432, and PMC-Sierra RM700. CodeWarrior tools can be used to debug code via a JTAG connection such as EJTAG 2.5. CodeWarrior IDE 4.0 is hosted on Windows 98, 2000, and NT 4.0.
