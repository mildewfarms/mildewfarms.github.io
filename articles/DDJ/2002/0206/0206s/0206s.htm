Dr. Dobb's Journal June 2002

Demeanor, from WiseOwl, is a code obfuscator for .NET applications. It secures .NET applications by making them difficult to reverse engineer. Demeanor also removes unnecessary information and obfuscates and shortens required symbolic names for loading/downloading of .NET assemblies. There are two versions of Demeanor available: Demeanor .NET Lite, which performs symbol and metadata obfuscation, and Demeanor .NET Enterprise, which performs these functions as well as control-flow obfuscation.

Pramati Technologies has released Pramati Server 3.0, the first application server to achieve J2EE 1.3 compatibility, and Pramati Studio 3.0, an integrated development environment for building J2EE applications. Pramati Server 3.0 carries a high-performance EJB 2.0 container and provides load balancing, failover, and hot deployment, as well as remote monitoring via web-based consoles. It also works with any J2EE-compliant development environment. Pramati Studio 3.0 is application server agnostic and supports direct deployment to several J2EE-compliant app servers, including those from BEA and Oracle. It also features a migration tool so that code created with it can be moved from one application server to another.

Sourcefire has announced the availability of the OpenSnort Management Console (OMC), an appliance-based sensor management and intrusion detection analysis tool. The OMC centralizes the security environment of network devices, distributed enterprise sensor deployments, and large service provider environments. The OMC allows management from a central point on the network, and its web-based GUI provides browser-independent local or remote access. Other features include: multisensor configuration management and data aggression, high-performance data management, encrypted communications with sensors, forensic data analysis, network configuration, signature management and creations, and real-time alert configuration.

HiT Software has released winAllora Xpress, a tool for importing/exporting XML data into/from relational databases without programming. WinAllora Xpress is a Windows application used to define relationships between XML document elements and relational database columns. An expression editor also allows detailed data conversion during transformations. Once winAllora Xpress mapping is in place, import/export XML-RDB transformations are a one-button operation. WinAllora Xpress supports industry standards including OLEDB/ODBC SQL middleware, W3C DTD and XML schema, and MSXML.

InCert Software has announced the availability of its Halo Application Fault Management Monitor, which monitors, pinpoints, reports on, and provides a source-code level, root-cause diagnosis of software failures in deployed applications. Halo watches over both proprietary and third-party applications, and sends an alert to a central operations console detailing the exact machine, application, module, and even line of source code responsible for the failure. Halo works as a standalone application or in conjunction with other systems management products. The software supports Windows, Solaris, and Java applications.

Sagantec has released SiFix, a tool for automatic correction and optimization of physical designs. With SiFix, you can implement layouts using simple rules and rely on SiFix to handle the additional complex rules. You can also begin physical implementation before final design rules values are fully available. SiFix scans through layout files, using quality and reliability rules to adjust width, spacing, and increase strapping in selected nets and segments. When it finds errors and hazards, it fixes them locally, leaving the rest of the physical design untouched. SiFix works equally well on full-custom layout or cell-based designs. It is available on Solaris, HP-UX, and Linux platforms.

TurboCache, from OptiView Technologies, is a web content and image optimization tool focused on reducing image bulk, caching, and accelerating all web traffic. TurboCache is a transparent caching proxy server that optimizes text and images, speeding page downloads across any connection including corporate LANs, modems, and wireless devices. It adds content optimization by reducing the byte size of content prior to caching. TurboCache optimizes web content without visually degrading images or text, in addition to caching traffic results to eliminate reoptimization and reduce latency.
