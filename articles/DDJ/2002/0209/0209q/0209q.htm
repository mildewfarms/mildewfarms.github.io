Dr. Dobb's Journal September 2002

By Della Song

Sophist Solutions has rolled out Version 3.0 of its Led C++ Class Library, which is designed to support the construction of text editors, word processors, HTML editors, and other text-intensive applications. The Led text engine supports style runs, paragraph layouts, bullet lists, word wrap, object embedding, and the like. The library is cross platform, supporting Windows 95/98/2000/ME/XP, Linux/X-Windows, and Macintosh.

Sleepycat Software has introduced Berkeley DB XML, an embedded open-source XML data management engine. Berkeley DB XML is built as a module on top of Berkeley DB, which provides a robust, scalable, transactional data management service for applications. It offers the same database management services as Berkeley DB coupled with native storage and query services for XML data, which lets you create web-based applications and services with unmatched reliability, scalability, and performance.

VeriFinger, from Neurotechnologija, is a comprehensive fingerprint-recognition technology that matches up to 10,000 fingerprints per second. It can be used for both fingerprint identification and fingerprint verification, requiring only 512 KB of memory with low-memory DSP and other embedded applications. VeriFinger integrates with existing security systems and works with all scanners, databases, and UIs. It is available as an SDK for Windows; source code is available for DSP-based applications, letting you port the system to other operating systems or embedded devices.

GrammaTech has released CodeSurfer 1.6, a maintenance, understanding, inspection, and analysis tool. Its program-analysis techniques precompute program properties allowing the quick and precise analysis of ANSI C source code. The new Version 1.6 features include the ability to inspect type information — a variable's type is now shown in the property sheet for that variable; the ability to surf to macro definition from their use in the code and a fully expanded macro; new functionality that lets you do queries on the points-to-set of actual parameters of functions; and enhanced scalability and build performance. CodeSurfer runs under Windows, Linux, and Solaris 2.5 and later.

Quest Software has released its Foglight 3.0 application monitor. Foglight monitors every layer of the application technology stack including web servers, databases, application servers, and operating systems. It monitors end-user, network, and disk space to maintain the performance and availability of the application infrastructure. Foglight 3.0 facilitates event management, reporting, real-time and historical diagnostics, and the maintenance of SLA and response time guarantees.

Rational Software has released Rational Suite v2002 Release 2, introducing Rational XDE Modeler, a new member of the Rational XDE family, and ScriptAssure, an industry-first capability in Rational Suite TestStudio. Rational XDE Modeler enables you to build visual models that combine UML with free-form drawing, and runs inside IBM WebSphere Studio Application Developer, IBM WebSphere Workbench, and Visual Studio .NET; and ScriptAssure produces resilient test scripts for Java and web business applications. New capabilities in existing Rational Suite products include the following: Rational XDE Professional now offers data modeling, web publishing, reporting, and assisted modeling capabilities; Rational PurifyPlus now provides access to memory leak detection, code coverage, and performance analysis from directly inside the IDE; and Rational Rose Real Time provides embedded software with passive state machine code generation capabilities.

LiveDevices has released Realogy Real-Time Architect (RTA) 3.1, a real-time operating system. RTA 3.1 is the first certified RTOS to provide full implementation of the latest OSEK OS 2.2 standard, which defines RTOS support for a wide range of embedded applications. It also provides integrated timing analysis allowing developers to identify where additional functionality can safely be added without violating timing requirements.

Prosentia has developed ProCrypt, which lets you control the number of machines on which customers install your software. It avoids the difficulties of dongles and copy protection because licensing is done via the Internet. Public/private key cryptography is used to store the licensing information on the customer's computer in a form that cannot be tampered with. The software consists of an ActiveX control that you embed in your application, and a set of Active Server Pages that you run on your web server. ProCrypt runs on all Windows platforms and requires that the application can contain ActiveX controls.
