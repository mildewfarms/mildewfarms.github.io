Dr. Dobb's Journal October 2002

The Kermit Project at Columbia University has released Kermit 95 2.0, a secure, scriptable, customizable terminal emulation and file transfer GUI for Windows. It comes with 40 different terminal emulations, including VT320, Linux console, ANSI, SCOANSI, HP, IBM, Wyse, and TVI; with colors, key mapping, 2-million-line scrollback, URL hot spots, and character set conversion (including Unicode). It also includes Kermit, FTP, and XYZMODEM file transfer; and numeric and alphanumeric paging. Any routine or complex communication or data transfer task can be automated through its built-in script programming language.

eXcelon has released Stylus Studio 4.0, a development environment for creating, validating, and debugging XSL stylesheets and XML-to-XML mappings. It contains a visual schema diagram editor that eases the creation and modification of XML Schema. It provides synchronized views of source XML Schema--a tree view and the XML Schema diagram--that enable you to work with XML Schema at multiple levels. In addition, Stylus Studio 4.0 supports Xalan-J-based debugging; and is the only XML/XSLT IDE with back mapping, which associates a defect with the related piece of XSLT source code to automate the process of debugging XSLT applications. Also from eXcelon is the Verity Search add-on for its native XML DBMS, the Extensible Information Server (XIS), which lets you integrate unstructured corporate information with structured XML data that can then be leveraged and extended by applications both inside the enterprise and beyond.

Parasoft has announced WebKing 3.5, a comprehensive testing tool that automates elements involved in developing, deploying, and maintaining reliable, multitier web sites. The new version automatically analyzes server log files, and then creates virtual users that mimic actual user paths and traffic patterns; it also introduces features that automate both SOAP testing and other test scenarios. Other new features include a CodeWizard for WSDL tools that validates WSDL; a Java test tool that allows testing of applets, .class files, .java files, and JSPs; and an HTML cleanup tool that fixes overlapping tags, invalid attributes, and problems with the overall document structure. WebKing 3.5 is available for Windows NT/2000/XP, Linux, and Solaris.

Cytron Research has released Visual VCL Pack1 for Delphi and C++Builder. It provides for over 30 VCL components and classes, 17 of which are design-time controls. The pack includes two development kits, one for designing custom comboboxes, the other for the creation of custom dictionaries and multilingual spell checkers. The ComboBox kit includes examples such as a Tree View combobox as well as a calculator combobox. It also comes with utility controls that is comprised of a binary streaming control, a stringlist streaming control, as well as a file storage control. In addition, the utility controls also include a 128-bit encryption control.

NTRU Neo for Java is a security system designed for all wireless Java devices and applications. It is optimized for a complete range of devices, and the toolkit includes the standard Neo for Java and JavaKE (Kilobyte Edition), a smaller version designed for space-constrained environments. NTRU Neo for Java delivers enhanced fraud protection, requires minimal processing power, and provides APIs that make it easy for you to include security in your wireless Java products. NTRU for Neo supports J2ME and Java virtual machines.

Altia has released Altia Design 5.0, a simulation graphics tool that helps you create custom and interactive virtual prototypes of products in development. Version 5.0 enhancements include an object browser for design navigation; support for transparent backgrounds within prototypes (which allows other windows to show through an Altia design); GUI-style prebuilt drag-and-drop components such as radio buttons, tabs, toggles, momentary buttons, and menus; and improved copy and past capabilities. Altia Design also assists in debugging, optimizing, and regression testing. The software connects to all leading system simulation tools and programming languages.

Octane8 4.0 enables the creation, deployment, and management of individual or multiple corporate web sites, intranets, extranets, or portals, along with enterprise-level applications. Other changes include multiple levels of folders and files within scripts; expanded check-in and check-out features; an enhanced table editor; a "save-as" feature for templates, scripts, and application files; simultaneous upload for multiple items and entire folders; enhanced user management; and page publishing by date.
