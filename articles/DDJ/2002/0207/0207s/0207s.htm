Extended Systems has shipped its Advantage Database Server TDataSet Descendant client for Kylix (Delphi for Linux). Kylix support lets Delphi developers port existing Advantage TDataSet Descendant applications to the Linux operating system without rewriting any of their database access code. The new Advantage Kylix components will provide database access without the use of Borland's dbExpress, just as the Advantage Windows components currently provide database access without the use of the Borland Database Engine (BDE).

Query Studio Developer, from Austin Sierra Technologies, integrates into the Microsoft Visual Studio .NET IDE and provides a pathway to deliver data to applications without a single line of code, enabling the build and management of multistep queries visually. Query Studio has a QBE-type interface, can preserve existing code, and exposes SQL script to ensure that any problem can be solved. Additionally, you can develop and deliver data in a variety of applications. Query Studio supports most ANSI-compliant SQL databases including Microsoft Access, Advantage, Apollo, Btrieve, DB2, Informix, InterBase, Microsoft SQL Server, MySQL, Oracle, and Sybase.

Empirix has released FirstACT 2.0, a web services testing tool that lets you create, execute, and maintain performance tests for individual web services. Key features in FirstACT 2.0 include a new diagnostics capability that lets testers analyze the accuracy of the WSDL file to ensure that end users can acquire the right information from the web service; improved, easy to use automatic test-case generation that permits users to test web services without programming; and an enhanced user interface that lets users implement the test solution with little or no training.

Virtuoso 2.7, from OpenLink Software, is a cross-platform, multipurpose, multiprotocol server for SQL, XML, and web services. This release enables the rapid creation, integration, hosting, and deployment of web services that coexist with existing IS infrastructures. Virtuoso enables the publishing of Persistent Stored Modules — SQL stored procedures, Java objects, and .NET objects — such as SOAP, WSDL, and UDDI-compliant web services. It also enables the creation of XML documents from SQL databases on the fly, such that XML-based access to disparate data can be used as the data foundation for current and next-generation web services. Virtuoso 2.7 is available for Windows, Linux, UNIX, and Mac OS X. It includes support for all ODBC and JDBC accessible databases.

The Standard Performance Evaluation Corp. (SPEC) has released SPECweb99_SSL, a benchmark that adds Secure Sockets Layer (SSL) Protocol support to SPECweb99, the worldwide standard for web server performance evaluation. SPECweb99_SSL is built on the same text harness and uses the same workload and file sets as SPECweb00. It tests secure web server performance using HTTP 1.0/1.1 over the SSL Protocol. It is an extension of SPECweb99.

Entrek Software has announced the availability of Entrek TOOLBOX Windows CE Edition 1.2. Included in the TOOLBOX are CodeSnitch, an application run-time error detection tool that pinpoints errors in your application such as memory and resource leaks, heap corruption, invalid parameters passed to APIs, and API failures; and Procman, which allows you to browse, monitor, and modify objects on your system. With TOOLBOX, you can browse the system hierarchy of processes, threads, windows, and modules; monitor items such as process virtual memory, heap, and CPU usage, or a thread's CPU usage; modify items such as a thread's priority or quantum, to help fine-tune an application; and debug and fix GUI problems such as Window's style bits, Z-order, and position. TOOLBOX supports Windows CE 2.11 and higher.

Excelsior JET 2.5 is a Java Virtual Machine replacement that transforms Java applications into conventional Windows executables (EXE and DLL files) containing functionally equivalent optimized native Intel x86 code. New in this release is the GUI-based control panel, which has a wizard-like interface that guides you through the process of JET project creation step by step. JET 2.5 is available for Windows 98/NT/2000/XP.
