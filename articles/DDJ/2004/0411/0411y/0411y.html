BlueGlue 3.0 (previously known as "Out-of-the-Box") from OpenLogic is a toolkit distribution of open-source development tools for Java and LAMP developers. It features more than 100 open-source projects, including Apache, JBoss, Tomcat, Struts, Eclipse, PHP, Python, MySQL, PostgreSQL, CVS, Hibernate, AspectJ, and wiki, with automated installation, configuration, integration, deployment, and testing of any combination of projects chosen. Regular updates keep everything current and sample applications are provided.

ceTe Software has released Version 3.0 of its DynamicPDF Generator for .NET and its DynamicPDF Merger for .NET. These tools are class libraries used to create and manipulate PDF documents programmatically. DynamicPDF Generator 3.0 for .NET lets you programmatically create PDF documents via an object model that is extensible and allows a high degree of customization. DynamicPDF Merger 3.0 for .NET lets you programmatically merge, split, stamp, form-fill, and personalize existing PDF documents.

ASNA is now shipping its DataGate Component Suite (DCS) for .NET, providing VB.NET and C# programmers with access to the iSeries platform. Performance gains are realized by using ASNA's own OS/400-based host server program rather than the OS/400-provided server program that OLE DB uses. The DataGate host server respects all server-side facilities, such as referential integrity and triggers, and obeys all OS/400-imposed user authorities and security. The DCS client component is 100 percent managed MSIL .NET assemblies and runs on either individual Windows desktops or Windows servers. Although DCS is targeted at VB.NET and C# programmers, it works with other .NET languages such as Delphi or C++.

NVIDIA has released several new developer tools. An updated version of FX Composer includes support for Shader Model 3.0 and many new sample shaders, as well as UI improvements, support for NV4x GPU performance analysis, and a beta SDK that lets you write your own automation scripts and importers/exporters. Version 2.0 of NVPerfHUD, previously available via NVIDA's registered developer program only, analyzes your graphics pipeline performance one stage at a time, and displays real-time statistics that can be used to diagnose performance bottlenecks in your Direct3D 9 application. Finally, the NVIDIA SDK 8.0 includes Shader Model 3.0 as well as hundreds of new code samples, GLSL and HLSL effects, and new tools.

Pro Fortran is fully compatible with Panther and all Apple tools and libraries, and is optimized for the G5 processor. DTM Soft has released the DTM Migration Kit, a Windows application that works with unified interfaces of ODBC and IDAPI as well as with the Oracle Call Interface. It is capable of importing data from plain text, Microsoft Access databases, Microsoft Excel spreadsheets, Paradox files, or DBF (dBase and FoxPro) files, and exports data to plain text, RTF format (compatible with Microsoft Word), HTML, or XML. It is also capable of creating sets of SQL statements. Data migration can be automatically performed at scheduled time intervals.
