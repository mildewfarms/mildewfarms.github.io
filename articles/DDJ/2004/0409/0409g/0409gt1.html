Step Procedure
1 Band-In-A-Box loads the song file and associates it with the export style.
2 Band-In-A-Box exports the tune as a Standard Midi Format (SMF) file.
3 Finale imports the SMF file and converts it to notation.
4 Finale exports the notation in MusicXML format.
5 ChordBuilder reads the MusicXML file that Finale exported.
6 ChordBuilder converts the data to lead sheet format and writes a new MusicXML file.
7 Finale imports the MusicXML file that ChordBuilder created and converts it to notation.
8 Finale saves the notation in its own file format.

Table 1: ChordBuilder procedures.
