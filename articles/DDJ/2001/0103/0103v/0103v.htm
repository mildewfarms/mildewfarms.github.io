VMware has announced two new servers: the GSX Server for Linux platforms and the ESX Server, which does not require an operating system. The GSX Server is intended to provide improved control on Intel servers through the use of virtual machines, all of which can be remotely managed. New features include the ability to run multiple applications on a single host; detect server crashes and automatically switch to a backup; create a small server farm dedicated to testing and integration; and use only one server to run in isolation sets of applications that were designed for separate machines. The ESX Server also runs multiple virtual machines on the Intel architecture and shares many capabilities of the GSX Server. It includes monitor architecture to provide scheduling, resource governing, memory management, high-performance I/O, data collectors, and an SNMP agent.

WebGain's TopLink for Java Version 3.0 is intended to bridge the gap between JavaBean objects and relational databases. TopLink abstracts the interface to the database and maps the data directly into application objects, so you can build object-based applications even when the database you work with is relational. TopLink is compatible with a number of application servers, including IBM WebSphere and BEA WebLogic. TopLink will also be included in the professional edition of WebGain Studio.

The Apache Software Foundation's XML Project has released the beta version of Batik 1.0, an open-source Scalable Vector Graphics (SVG) toolkit. The project provides a set of core modules that can be used together or individually to support specific SVG solutions. Example modules include SVG parsers, SVG generators, and SVG DOM implementations. Batik is also designed for extensibility, and lets you handle custom SVG tags. The toolkit also includes a full-fledged SVG Viewer implementation. Batik is compatible with any operating system that supports Java.

TECA has released LoadBlobPro 1.0, a fully graphical import-and-export utility for adding Binary Large Objects to a SQL server database. LoadBlobPro eliminates the need to write custom file import and export programs, and is able to import and export most PC file types — including image, audio, executable, and application files — to and from a SQL server image column. An integrated file system explorer allows navigation of the PC-file system while a built-in database viewer supports interactive selection of the image column. In addition, LoadBlobPro can perform set selection, joins, and updates of record sets using custom SQL select statements.

Reliable Software is offering Code Co-op 3.0, a serverless version-control system for collaborative development. Code Co-op is a fully distributed system requiring no network connection: Synchronization between project members is handled through the exchange of scripts using LAN or e-mail. Various automation features have been added, including autosynch and automatic retrieval of synchronization scripts. Code Co-op is available for Windows.

Computer Access Technology Corp. has developed Merlin's Wand, a test generator for the Bluetooth wireless protocol. Merlin's Wand is an intelligent Bluetooth device that can be used as a verification/validation tester or as an engineering and analysis tool. Its combination of hardware and Windows-based application software is capable of acting as a standard Bluetooth master or slave device within a piconet. Merlin's Wand can receive data within the piconet or issue individual protocol commands and test sequences to a test device. When used in conjunction with CATC's Merlin protocol analyzer — a nonintrusive tool for capturing, displaying, and analyzing Bluetooth piconet data — Merlin's Wand can be controlled with the test device through a single interface, allowing for real-time capturing of the results of test sequences.
