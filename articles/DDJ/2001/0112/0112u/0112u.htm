PolarLake has launched its PolarLake 1.0, an XML platform for Java that supports the development of applications that process XML documents. PolarLake 1.0 provides a set of system and network services combined with a rules-based programming framework. It also includes a set of development tools that supplement standard Java development tools. PolarLake is entirely Java based and supplements the J2EE platform if required. It can optionally interoperate with Microsoft's COM platform.

Sun Microsystems has released Version 3.0 of its Forte for Java IDE. The company has also contributed seven modules to netbeans.org. The Forte for Java IDE is built on the NetBeans platform. A new web services module provides automatic binding of Java technology and XML through wizards, generation of Java code that supports run-time access and execution of services, and support of web-based publishing to a services registry. The open-source modules include an external editor, C/C++ and Fortran support, XML support, a database explorer, CORBA support, file copy, and terminal emulation. Also new from Sun is the J2EE 1.3 platform specification, released in conjunction with a reference implementation (RI) and a Technology Compatibility Kit (TCK). J2EE connectors enable integration with existing systems.

Innovative Routines International has announced the availability of CoSORT for the IBM eServer iSeries. Zend Technologies has announced its Zend Accelerator 2.0, a server-side caching program designed to speed performance of PHP-driven web applications by cutting web site latency. The company claims that the software lets PHP-based web sites handle more page views and transactions, serve more concurrent users, deliver faster responses to user requests, conserve an organization's computing resources, and reduce new hardware requirements. Users can customize and manage Accelerator features, and its code optimizer performs a multipass optimization for more efficient code utilization. Zend Accelerator is available for Linux, Solaris, and FreeBSD.

It is supported on Windows, AIX, HP-UX, Solaris, and Linux.

Ricoh Silicon Valley has announced availability of eCabinet 2.1, which is designed to easily save, organize, archive, and share information and electronic filing in document-intensive environments. Attached to existing network infrastructures, eCabinet communicates with network peripherals — copiers, scanners, fax machines, printers, and even PCs (including web and e-mail content) — storing files without creating network bottlenecks. Saved information is automatically indexed and can be routed to users and workgroups, making search and retrieval of critical business documents a straightforward process. Version 2.1 includes settings for French, Italian, German, Dutch, and UK English, 17-language OCR support, and a downloadable universal print driver.
