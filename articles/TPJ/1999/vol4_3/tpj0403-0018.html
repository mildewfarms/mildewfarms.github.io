TPJ One-Liner #42

Extract unique elements from a list given a key function 
sub unique (&@) {
  my($c,%hash) = shift;
  grep { not $hash{&$c}++ } @_
}
@list = unique { $_       } @list;  
# Remove duplicate strings from @list.
@obj  = unique { $_->name } @obj;
