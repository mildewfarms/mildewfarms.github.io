New Products

Pedestal Software Announces SecurityExpressions

Pedestal Software Inc. announced the availability of SecurityExpressions 3.0, a solution for deploying and managing enterprise system security policies. According to the company, with SecurityExpressions 3.0, enterprises have an automated, agent-less solution to audit, deploy, and enforce security policies across servers, workstations, and desktops in a fraction of the time previously required. SecurityExpressions 3.0 can also help reduce the cost and time to implement and manage system security policies that control items such as password rules, user permissions, and registry settings for corporate and regulatory compliance. SecurityExpressions 3.0 is available immediately. Pricing is based on the number of systems scanned by SecurityExpressions and starts at $495 for servers and $30 for workstations and desktops. For additional information on SecurityExpressions, visit: http://www.pedestalsoftware.com.

For more information, visit: http://www.moonlight.com.

The AlterPath ACS makes possible convenient, secure, and remote access and management of all console ports in a data center at any time and from anywhere, even when the network is down. TOLIS Group Announces BRU CLI

The TOLIS Group, Inc. announced the availability of BRU CLI (Command Line Interface) data protection software for Mac OS X systems. According to the company, BRU CLI fulfills the need for easy-to-administer, reliable headless backup on Apple's Xserve platform. BRU CLI includes TOLIS' tapectl and libctl tools, which provide control support for tape drives and libraries under Mac OS X. BRU CLI provides headless Xserve backup and restore processes remotely via ssh or telnet access and through scheduled scripted operation for fully automated lights-out backup. Users of Apple systems with available monitors and keyboards running on Mac OS X 10.2 and newer can administer BRU CLI locally. BRU CLI backs up and restores file elements including data fork, resource fork, finder info, and UNIX permission attributes. BRU CLI is priced at $329 and is available immediately. The free, full-function 30-day demo can be downloaded from TOLIS' Web site at: http://www.tolisgroup.com.

TFS Introduces TFS UnixControl

TFS Technology (TFS) announced TFS UnixControl, a management solution for controlling access to UNIX-based information assets, with performance capabilities in heterogeneous, multi-vendor UNIX environments. According to the company, TFS UnixControl can help organizations define, enforce, and audit a security policy, as well as assist in management of the identification and authentication of users, creation of user groups and roles, establishment of access control policies, assurance of data privacy and integrity, and auditing the entire process, from a central security management console. While the TFS UnixControl centralized management tools simplify user administration, all basic characteristics of the UNIX system remain the same; user accounts on all managed hosts are synchronized and local etc/passwd files are updated. Full integration with other management tools such as NIS/NIS+ is also included. For more information, visit: http://www.tfstech.com.
