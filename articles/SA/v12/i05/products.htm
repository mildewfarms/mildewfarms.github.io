New Products

Permeo Application Security Platform 4.1 Released

Permeo Technologies announced the immediate release of the Permeo Application Security Platform v4.1. According to the company, the new version can enable businesses to extend application security to wireless networks and provide an increased level of control through application-level filtering. Permeo v4.1 provides increased application security for WLAN deployments by protecting all application traffic in full strength SSL while allowing users to transparently secure their applications in any protected wireless zone. Permeo allows the enterprise to leverage its existing authentication and authorization services while extending application-level security to wireless users.

For more information, visit: http://www.permeo.com.

Redline Networks Introduces E|X 3250 Enterprise Web I/O Accelerator

Redline Networks recently announced a new data center appliance that provides secure, accelerated Web application access across the enterprise. According to the company, Redline Networks' E|X 3250 Enterprise Web I/O Accelerator is a Layer 7 data center appliance that automatically enables SSL security, accelerates application transaction performance, and enables existing equipment to handle greater transaction capacity with increased reliability.

For more information, visit: http://www.RedlineNetworks.com.

Lycoris and Ericom Software Deliver Desktop/LX InterConnect

Lycoris and Ericom Software recently released Desktop/LX InterConnect, a corporate desktop with full office suite and host connectivity tools. According to the company, Desktop/LX InterConnect features Ericom Software's native Linux PowerTerm InterConnect software, the Lycoris ProductivityPak office suite, and the Lycoris Remote Desktop Client.

For more information, visit: http://www.lycoris.com.

Rainfinity Announces RainWall High Availability Platform

Rainfinity recently introduced the RainWall High Availability Platform. According to the company, instead of using disparate technologies to build redundancy into each Internet gateway application, the RainWall High Availability Platform provides integrated fault tolerance, acceleration, and centralized management that increase resiliency while minimizing implementation, management, and support costs. RainWall High Availability Platform is available today with RainWall and RainConnect 3.1.

For more information, visit: http://www.rainfinity.com.

Snap Appliance Increases Storage Capacities of NAS Appliances

Snap Appliance, Inc. announced it has increased the storage capacities of all models of its Snap Server 1100, 2200, and 4100 series by as much as 100%. According to the company, the enhanced SnapOS operating system simplifies file serving and client backup, and is available to existing Snap Server customers via free download.

For more information, visit: http://www.snapappliance.com.

Packeteer Releases PacketShaper Xpress

Packeteer, Inc. recently announced PacketShaper Xpress, the new version of their application traffic management system. According to the company, PacketShaper Xpress adds traffic acceleration to speed the performance of networked applications and extend bandwidth resources.

For more information, visit: http://www.packeteer.com.

Secure Shell Toolkit 4.0 Extended

SSH Communications Security announced the extension of its SSH Secure Shell Toolkit 4.0, for use within network infrastructure products, such as routers, switches, and network appliances. According to the company, SSH Secure Shell Toolkit 4.0 has been optimized for embedded systems, enabling manufacturers to incorporate SSH Secure Shell technology into existing remotely managed devices.

For more information, visit: http://www.ssh.com.

Appro Introduces HyperBlade B221X

Appro announced its HyperBlade cluster solution. According to the company, the HyperBlade B221X cluster will feature a combination of 16 nodes Linux cluster-running MPI over InfiniBand 4x fabric provided by InfiniCon Systems and Lane15 Software.

For more information, visit: http://www.appro.com.

Metagenix Announces MetaPure Data Quality Auditing Tool

Metagenix released MetaPure, a data quality auditing tool for enterprise database systems. According to the company, MetaPure sits between the data flows entering a data system and the data system itself, continuously monitoring and auditing data to ensure it adheres to pre-configured, customized business rules.

For more information, visit: http://www.metagenix.com.
