New Products

Altiris Releases New Altiris® Server Provisioning SuiteTM

Altiris, Inc. has announced the new Altiris® Server Provisioning SuiteTM to automate time-consuming manual management tasks and promote consistent server system states. Designed to simplify server deployment and configuration management, Altiris Server Provisioning Suite includes new capabilities for patch management, operational state backup and recovery, and performance and availability monitoring for dynamic provisioning, all from a centralized Web console.

Altiris Server Provisioning Suite will be available in July 2003. The suite is offered at three different levels of functionality ranging in price from $300 to $900 per server. Additional volume discounts are available. For more information, visit: http://www.altiris.com/.

iConductorTM Improves its Apache Server Module

Farpointer Technologies, Inc. has announced new features for its iConductor Apache Server Module that provides easy to implement security and access control for Web site directories and files. According to the company, users can implement iConductor's comprehensive access control features without using complicated LDAP servers. Access to directories or files can be controlled globally at the directory level all the way down to the file content level. Additional security levels may be ascribed to any file or portions of a file to allow or disallow instructions contained therein. A shareware version of iConductor and an online demonstration are available at: http://www.i-conductor.com.

VanDyke Software Releases VShell 2.2 for Unix

VanDyke Software announced the public beta release of version 2.2 of its VShellTM SSH2 server for Windows and now Unix platforms, including Red Hat Linux and Sun Solaris. VShell for Unix offers precise and granular control over end user privileges, automation tools to reduce administrator time spent on routine tasks, and industry-leading support for authentication methods favored by enterprise organizations. VShell 2.2 is the first Secure Shell server to offer built-in support for Kerberos (v5) authentication via GSSAPI.

VShell uses Access Control Lists (ACLs) to align network access, file transfer, and application tunneling privileges with a company's security policies. For more information about VanDyke Software, visit the company's Web site at: http://www.vandyke.com.

Aspen Releases Version 2.0 of ABC Management Software

Aspen Systems, Inc. announced version 2.0 of their turnkey, browser-based software management system, Aspen Beowulf Cluster (ABC) Management Software. Aspen Systems has expanded ABC 2.0 to include: support for AMD OpteronTM processors, SSL enabled access, shadow copy/restore capabilities, diskless node configuration, administration and management, expanded plugins, support for heterogeneous cluster nodes, expanded node self-reporting statistics, SNMP trap rendering sort and database purge capabilities, and simplified installation.

Aspen Systems ABC 2.0 is available immediately. Pricing for ABC starts as low as $100 per node. For more information, visit their Web site at: http://www.aspsys.com.

Veritas Announces Global Cluster Manager 3.5.1

Veritas Software Corporation announced the latest version of Veritas TM Global Cluster Manager software that allows administrators to centralize control of their disaster recovery environment for application availability and data replication products from EMC, Hitachi and Veritas -- on all major platforms, including AIX, HP-UX, Linux, Solaris and Windows. Veritas Global Cluster Manager also now automates site-to-site disaster recovery for Exchange environments, helping to maximize the availability of corporate e-mail. Veritas' latest disaster recovery and availability management software helps heterogeneous IT departments provide a foundation for the delivery of IT services as an accountable, always-on utility.

New enhancements to Veritas Global Cluster Manager include: Wide-area disaster recovery for Microsoft Exchange 2000 servers, support for management of Hitachi TrueCopy, Hitachi's data replication suite, support for the IBM AIX operating environment, and management of clustered Linux servers. Pricing for the new Veritas Global Cluster Manager starts at $1,495 per site. For more information, see: http://www.veritas.com.

CA Updates BrightStor Enterprise Backup

Computer Associates International, Inc. (CA) announced the availability of BrightStor Enterprise Backup 10.5, a backup and recovery solution that provides policy-based management backup functions that allow storage administrators to assign virtually every type of data across the enterprise to any available and appropriate storage resource.

According to the company, BrightStor Enterprise Backup gives storage administrators the flexibility to use the resource most appropriate for each storage type by integrating with CA's own popular management solutions -- including BrightStor ARCserve Backup -- and with a wide range of leading third-party offerings including: Iron Mountain and other offsite vaulting services and snapshot image technology from EMC, Hewlett-Packard, Hitachi, and Microsoft. BrightStor Enterprise Backup also incorporates BrightStor Portal technology to provide common management and cross-server reporting for all enterprise storage resources.

BrightStor Enterprise Backup allows policies to be defined from a centralized console across all major platforms. For more information on CA, please visit: http://ca.com.
