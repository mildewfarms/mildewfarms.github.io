New Products

Gibraltar Software Releases Everguard System 2.0

Gibraltar Software, Inc. recently announced the second major version release of its cross-platform patch management product, the Everguard System. According to the company, the Everguard System 2.0 adds automated patch deployment capability to the reporting tools found within Everguard 1.0, providing systems administrators with a hands-off method of applying patches to the latest vulnerabilities. Gibraltar Software allows admins to address vulnerabilities within the Windows, Red Hat Linux, and Solaris operating systems. Everguard 2.0 is ideal for those companies who have deployed multiple operating systems across their networks.

For more information, visit: http://www.gibraltarsoft.com.

Bradmark Announces Integrated Monitoring Solution

Bradmark Technologies, Inc. recently announced the completion of a 10-year investment in the development of an integrated interface for database administrators to monitor Oracle, Sybase, Microsoft SQL Server, and IBM DB2 databases. According to the company, NORAD is a fully integrated suite of tools designed to help the DBA monitor and maintain a single relational database environment or an enterprise of hundreds of disparate servers and databases. When combined with NORAD DBControl Online, the full functionality of NORAD can help maintain databases online, ensure the highest level of availability, and keep systems optimized at all times with one product. In a large heterogeneous environment where many different database applications, relational database technologies, and server platforms co-exist, a product like NORAD can be an asset to the DBA.

For more information, visit: http://www.bradmark.com.

MTI and Quantum Provide Integrated Backup/Recovery Solutions

MTI Technology Corp. and Quantum Corp. teamed to offer an integrated backup/recovery solution incorporating the Quantum DX30, a new high-capacity disk-based enhanced backup system that emulates a tape library. According to the companies, for this latest collaboration with Quantum, MTI has packaged a turnkey solution called DataSentry VTL (virtual tape library) that includes professional services, software integration, installation and maintenance. MTI's DataSentry VTL can help protect investments in backup procedures and equipment, enabling an enterprise to continue traditional tape operations for off-site storage and long-term archiving using its existing tape libraries. Together, MTI's management processor and Quantum's DX30 provide a full-service backup platform. The Quantum DX30's performance allows the unit to initiate file restores within seconds and recover a 280-gigabyte application server in less than an hour.

For more information, visit: http://www.mti.com.

Data Junction Announces Real-Time Integration Product

Data Junction Corporation announced the general availability of Integration Architect and enhanced Integration Engines to handle distributed integration for enterprises. According to the company, Data Junction now offers a single integration platform designed to grow and scale for integration projects of all sizes. Data Junction also announced that it has moved to a completely open XML-based metadata repository for managing and exchanging data and their corresponding integration processes. Integration Architect creates single integration processes that will integrate data from structured record sets (ASCII, ISAMS, etc.), documents with canonical form (EDI, XML, etc.), and unstructured data (reports, email, etc.).

For more information, visit: http://www.datajunction.com.

Procom Introduces ProMobile

Procom Technology recently introduced ProMobile, a backup system for notebook and desktop computers that can also be used for external storage. According to the company, the portable, pocket-sized system is available with capacities of 20, 30, 40 and 60 gigabytes and is a complete software and hardware solution that simply plugs into the computer's USB interface. It automatically scans the source drive, copying only those files that have been changed or newly created since the last backup was performed. ProMobile enables users to access lost data files, in native format, and to keep working even if the computer fails. With a compression option, drive space can be further optimized.

For more information, visit: http://www.procom.com.

INNOVATION Data Processing Announces S/390 Linux Rescuer

INNOVATION Data Processing recently announced S/390 Linux Rescuer, a disaster recovery solution for S/390 Linux. According to the company, S/390 Linux Rescuer allows FDR/UPSTREAM to recover a user's entire system, including their Linux system boot disk, file systems, device drives, and all their application program and data files. FDR/UPSTREAM 3.2.1 will include S/390 Linux Rescuer as a standalone system recovery facility for SuSE S/390 Linux systems running on IBM S/390 and zSeries.

For more information, visit: http://www.innovationdp.fdr.com.
