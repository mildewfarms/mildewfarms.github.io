Listing 6: mon_dsk.dat

#
# Data file for "mon_dsk"
#
# Entries are of the form:
#
#   FS    WN   SV   CR   FA
#
#  Where:
#
#   FS is the pathname of a filesystem to
#      be monitored
#   WN is the integer value which
#      indicates a "WARNING" level alarm
#   SV is the integer value which
#      indicates a "SEVERE" level alarm
#   CR is the integer value which
#      indicates a "CRITICAL" level alarm
#   FA is the integer value which
#      indicates a "FATAL" level alarm
#
#  Note: these numeric values represent
#  percentages of disk space
#
#  Example:
#
#   /var    70    75    80    85
#
########################################

# End of File
