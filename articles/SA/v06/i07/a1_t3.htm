Table 3: ACUA's programs

acua_dumpstats - dump statistics information for report generation.
acua_stats - show usage statistics in a human-readable format.
user_add - create a new UNIX/ACUA user account.
user_addrec - create a new ACUA user record.
user_ban - ban a phone number from the system.
user_bytes - find out who is transferring the most data.
user_daysleft - show number of days remaining until expiry.
user_data - show an ACUA user record in a human-readable format.
user_del - delete a UNIX/ACUA user account.
user_delrec - delete an ACUA user record.
user_dump - dump ACUA user record(s) for report generation.
user_expire - expire accounts, subscriptions and bans.
user_foreach - execute a command for each user in the passwd file.
user_kick - kick a user off the system.
user_login - authorize a user login.
user_mod - modify an ACUA user record.
user_pp - perform ACUA preprocessing on a file.
user_purge - purge dead accounts.
user_renew - renew user time/data limits.
user_subscribe - give a subscription to a user.
user_sync - synchronize the acua_users file with the passwd file.
user_timeleft - show remaining time in current session.
user_touch - exempt a user from the effects of a user_purge.
user_unban - unban a phone number from the system.
user_unsubscribe - unsubscribe a user.
user_updated - monitor system use and kick users off as needed.
clean_proc - kill processes left behind by users.
clean_tmp - unlink files left behind by users.
user_verify - perform call-back verification.
