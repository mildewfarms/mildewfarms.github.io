New Products

SmoothWall Announces Version 2 of Advanced Firewall

SmoothWall has announced the latest version of Advanced Firewall, its multi-function network security solution. New additions in this version include automatic failover to a standby system, improved load balancing, VoIP gateway protection, and spam filtering. The product is based upon a security-hardened version of Linux, and according to the company, software installation takes less than 10 minutes, with subsequent configuration and management done from any Web browser using an intuitive Ajax-enhanced GUI. For more information, visit: http://www.smoothwall.org/

Bakbone Delivers NetVault: Replicator 5.0

BakBone Software has announced NetVault: Replicator version 5.0, a real-time data replication software solution. NetVault: Replicator 5.0 incorporates new features that streamline data protection management and help customers prepare and plan for disaster recovery scenarios. Enabling bi-directional operations and flexible configuration options across Linux, Solaris and Windows platforms, this latest version replicates data and distributes content between multiple distributed sites over any IP network.

For more information or to download a fully functional 30-day evaluation copy, please visit: http://www.bakbone.com/products/downloads/.

Red Hat Enterprise Linux 5 Beta 2 Now Available

Red Hat has announced the availability of Red Hat Enterprise Linux 5 Beta 2. According to the company, Red Hat Enterprise Linux 5, to be released in early 2007, will provide the latest open source technologies for improved performance, security, and flexibility. Red Hat Enterprise Linux 5 is now available in Beta 2, and it incorporates a new, fully integrated server and storage virtualization solution. This release enables an integrated virtualization solution by coupling server virtualization with Red Hat’s clustering support. Download Red Hat Enterprise Linux Beta 2 from: http://www.redhat.com.

StoreVault Enhances StoreVault S500

StoreVault, a NetApp division, has announced new features to StoreVault S500 for improved simplicity, scalability, and data management for small and medium businesses (SMBs). StoreVault has added the following new features to Data ONTAP StoreVault Edition and StoreVault Manager:

  • StoreVault Replication. Based on NetApp enterprise technology, StoreVault Replication can copy both file and block data between locations. It is enabled with a software license key and is designed to simply manage replication between multiple StoreVault S500s using the new StoreVault Manager 2.0 interface.

  • Fibre Channel (FC) Support. StoreVault is teaming with QLogic Corporation to provide the StoreVault FC Starter Kit, which includes a factory-installed host bus adapter (HBA), a 10-port 4Gbit Fibre Channel switch, and QLogic SANsurfer Express software. For more information, visit: http://www.storevault.com

    Opsware Releases Opsware Orchestrator

    Opsware Inc. has announced Opsware Orchestrator, which adds Run Book Automation (RBA) capabilities, specifically workflow and orchestration, to its Opsware System 6 suite. RBA gives IT organizations the ability to automate the process of designing, creating, orchestrating, administering, and reporting on workflows used to control IT operations across the data center. Opsware Orchestrator’s new orchestration and process automation capabilities are provided through a partnership agreement with iConclude, by which iConclude’s RBA software is available immediately from Opsware, integrated with Opsware System 6.

    For more information, please visit: http://www.opsware.com.

    Free Qualys Security Scan Available for the New SANS Top 20

    Qualys, Inc. has announced the availability of a free network scanning service to help companies find and eliminate vulnerabilities listed in the annual SANS Top 20. The SANS Top 20 is designed by the SANS Institute and security experts from industry and government to provide organizations with a prioritized list of newly discovered exposures to their networks. Qualys’ free scan for the 2006 SANS Top 20 is available at: https://sans20.qualys.com.

    Technalign, Inc. releases Pioneer Linux in 32 and 64 bit versions

    Technalign, Inc. has released its new Pioneer Linux operating system to the community. The first release is the 32-bit version to be followed immediately by the 64-Bit version. Both Pioneer 32 and 64 include Firefox 2.0, Thunderbird, and Kmail email clients, OpenOffice.org, and many others such as K3b for CD/DVD burning. Technalign has partnered directly with Canonical and is using Kubuntu/Ubuntu as the base. Pioneer is a derivative of Ubuntu and not a fork. Pioneer Linux is targeted at new Linux users as well as experienced Linux users, and provides compatibility with many existing, proprietary applications.

    For more information, visit: http://www.technalign.com or http://www.tapioneer.com.
