Levanta Launches Intrepid X Linux Management Appliance

Levanta announced the release of the Intrepid X Linux management appliance. The Intrepid X caters to large Linux departments and datacenters with high scalability and mission-critical requirements. The Intrepid X features include on-demand functionality, disaster recovery, system portability, complete change control, unattended active/passive failover, and interoperability with iSCSI or Fibre Channel SANs for RPM-based Linux environments.

According to the company, with the Intrepid X, systems administrators can ensure maximum uptime and easily and securely manage their mission-critical Linux environments -- both physical and virtual. The Intrepid X supports Fibre Channel and iSCSI-over-Ethernet storage devices. For more information, please see: http://www.levanta.com.

< Aptitune Releases ZoneHound for Solaris SPARC

Aptitune Corporation has released ZoneHound for Solaris SPARC. The purpose of ZoneHound is to monitor the resource utilization of each Solaris Zone on a system with multiple zones.

The output of ZoneHound includes statistics for each zone based on data that is collected by the ps utility's -Z and -y options. A summary-based, user-specified sampling is then produced identifying the zones and memory usage. ZoneHound may be run as a standalone utility or together with Aptitune's SarCheck performance analysis tool. For more information, visit: http://www.zonehound.com.

Unisys Enhances OASIS Open Source Suite

Tripwire Unveils Tripwire Enterprise 6.0

Tripwire has announced version 6.0 of Tripwire Enterprise, featuring a variety of advanced reconciliation capabilities that provide greater control over unauthorized configuration changes, thereby increasing operational efficiency and mitigating risk.

Tripwire Enterprise 6.0 detects all configuration changes across the IT infrastructure and automatically identifies unauthorized change, policy exceptions, and security violations. Tripwire Enterprise 6.0 gives customers confidence that all unauthorized configuration changes will be quickly discovered, investigated and minimized, or eliminated as process maturity improves.

To ensure system integrity, Tripwire provides visibility to all change in the enterprise. It is the only solution to provide a single source for detecting all change across servers, network devices, databases, and directory services. By giving customers real-time visibility, Tripwire enables operational efficiency through reduced outages, faster recovery times and less unplanned work.

A free trial of Tripwire Enterprise can be downloaded directly from: http://www.tripwire.com/downloads/enterprise/request.cfm. For more information, visit: http://www.tripwire.com/.
