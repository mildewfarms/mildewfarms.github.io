This month's theme is Linux, and in this issue, Mark Roth shares some tips to keep in mind when upgrading or migrating Linux distributions, Ryan Matteson describes tools and techniques for troubleshooting some typical Apache Web server problems, Peter Arreman compares the features of various OpenSolaris distributions to those of Linux, and Mulyadi Santosa and Fawad Lateef provide an overview and comparison of Linux disk I/O schedulers.

Also, this issue includes our long-awaited supplement on virtualization, featuring articles that explain IBM's virtual I/O server technology, describe micro-partitioning on the IBM p5 595 server, and compare the pros and cons of some common virtualization solutions. Next month, look for articles on using the Online Certificate Status Protocol with Apache Web server and identity and data authentication with OpenPGP, among others.

If you don't already subscribe to the Sys Admin or UnixReview.com newsletters, I suggest checking them out. New items on UnixReview.com include a self-test on PHP from Emmett Dulaney, a review of Sams Publishing's Apache Phrasebook from Cameron Laird, and a review of Up.Time 4 browser-based monitoring software.
