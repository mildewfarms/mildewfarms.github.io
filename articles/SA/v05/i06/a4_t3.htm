Table 3: A summary of file system operations under UNIX and Windows NT

Operation UNIX Windows NT
View Owner ls -l File Manager/Security/Owner or CACLS filename
View Group ls -l No "group ownership"
View Permission ls -l File Manager/Security/Owner
Change Ownership chown Give new owner Full Control login in new owner and File Manager/Security/Owner/Take Ownership
Change Group chgrp Not available
Change Permissions chmod File Manager/Security/ Permissions/ (Add, Remove) or CACLS
Mount filesystem mount File Manager/Disk/Connect Network Drive
Share filesystem edit /etc/dfs/share and shareall* File Manager Select partition or subdirectory Disk/Share as

*This is for Sun Solaris. See UNIX System Administration, p.370 for other versions of UNIX.
