Figure 2: Installation procedure

1) Software Installation
- ensure correct patches are installed for appropriate OS
- ensure all parts of PPP packages are installed
2) Install and configure modems
3) Use the tip command to verify modem connectivity
4) Edit client and server PPP files to reflect desired setup
5) Use the cu command to verify UUCP portion of PPP
6) Start PPP with a high debug level and monitor log files
7) Modify configuration to reflect high speed serial cards
8) Use tip and cu again to verify connectivity
9) Startup PPP daemons
10) Cold start both client and server systems and monitor for success!
