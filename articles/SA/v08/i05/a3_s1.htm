Sidebar 1: Software Resources

Here are the Web sites of all the packages and commercial products mentioned in the article. These locations can be used to download software and get more information about these products. Some sites also include man pages and installation procedures for these tools.

Squid can be obtained from any of following sites:

http://squid.nalanr.net
ftp://squid.nlanr.net

The Squid Web site also contains a wealth of installation and configuration information. Squid manuals are also available from this site.

The tproxy package can be obtained from:

ftp://ftp.nlc.net.au/pub/linux/www/
Netscape proxy server (trial version) can be obtained from the Netscape Web site:

http://home.netscape.com
Microsoft proxy server:

http://www.microsoft.com
The Cisco Web site contains a lot of information about Cisco products:

http://www.cisco.com
The latest version of the Linux kernel can be downloaded from Sunsite:

http://sunsite.unc.edu
