Sidebar 1 : Recommended Sites

http://www.frforum.com
The main repository on the Web for frame relay information.

http://pierce.eosc.osshe.edu/frame.html
Provides an excellent analysis of frame relay technology.

http://scis.nova.edu/~raciti/frame_1.html
How frame relay can benefit the enterprise.

http://www.sangoma.com
For information about Sangoma and their line of PC WAN cards. Also home to an excellent collection of networking tutorials.

ftp://ftp.sangoma.com/pub
Sangoma's ftp repository.

http://www.cyclades.com
Information about Cyclades and their excellent line of serial multiport controllers and other Linux-supported products.

http://sunsite.unc.edu/mdw/LDP/nag/nag.html
The Linux Network Administrator's Guide (online). All about networking with Linux.

http://sunsite.unc.edu/mdw/HOWTO/NET-3-HOWTO.html
The Linux NET-3-HOWTO (online). Information on configuring the native kernel frame-relay driver for Sangoma boards.

http://cesdis.gsfc.nasa.gov/linux/beowulf/beowulf.html
The home page for NASA's Beowulf Project, a developing specification and software system for building Linux-based superscaling parallel computers out of "off-the-shelf" PCs. This site is home to a large amount of Linux networking codes setup and diagnostic tools for some 3Com and NE2000 compatible network cards, and a host of updated network drivers.
