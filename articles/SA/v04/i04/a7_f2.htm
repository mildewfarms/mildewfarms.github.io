# Some sample entries:

HCU ttyaw,M - 19200 hayes96
NCU ttyaw,M - 19200 hayesNO
SCU ttyaw,M - 19200 hayes
ACU ttyaw,M - 19200 hayes12
ACU ttyaw,M - 19200 hayesslow
#
#  The following entry is for use with the numpager script
#  The cu line as executed will look for an ACU entry with
#  a speed of 2400.  This speed must be unique for all ACU
#  references so that the hayespag Dialers entry is selected.
#  This speed MUST also be one that the modem will understand.
#
ACU ttyaw,M - 2400 hayespag
#

HCU ttyba,M - 19200 hayes96

SCU ttyba,M - 19200 hayes

ACU ttyba,M - 19200 hayesslow
#  Similar entry as above, but on a different port.
