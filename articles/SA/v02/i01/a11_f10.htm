Figure 9: The uuto command

uuto [options] file(s) system!userid

options
-m      Notify the sender through mail when the transfer
is complete.

-p      Copy the file(s) into the spool directory before
sending them to the remote system.

file(s)                 The pathnames of one or more files which are to
be transferred to the remote system.

system                  The name of the remote system.

userid                  The name of the user on the remote system who
is supposed to receive these files.
