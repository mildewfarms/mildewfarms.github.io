Figure 1: UUCP Files

Some of these files
are Version 2 or HoneyDanBer specific, and so will not have corresponding
files in the other version.

Version 2 Files         BNU/HDB Files                   Description

L-devices               Devices                 This file contains a
description of the devices
which are attached to
the system.

L-dialcodes             Dialcodes               You can use text to
designate areas instead of the
actual area codes.  This
file describes that text.
Dialers                 The file contains the
commands needed to get a modem
or other device to make a
call.
Maxuuscheds             This file contains a number
which is the maximum number of
uuscheds which can be running
at one time.
Maxuuxqts               The same as Maxuuscheds,
but it is for uuxqt.

USERFILE                Permissions             This file contains the
needed information to control
the security of your machine.
Poll                    You can elect to call sites
at specific times, which is
polling.  This file
describes who is called,
and when.
Sysfiles                Used to configure separate
Systems files for cu and uucico.

L.sys                   Systems                 This file contains the list
of systems we can call, or
that we know about, and how to
contact them.  It contains
phone numbers, user names, and
passwords.
Uutry                   Shell command to test chat
scripts by calling the remote
system.
remote.unknown          Shell command to record
calls from unknown systems.
uucheck                 Checks the UUCP directories
and Permissions file.

uucico                  uucico                  The UUCICO program

uuclean                 uucleanup               General cleanup tool.
uudemon.admin           Shell command to send UUCP
status information to the
system administratior.
uudemon.cleanu          Shell command to do general
UUCP system cleanup tasks.
uudemon.hour            Shell command to run the
UUCP scheduler.
uudemon.poll            Shell command to call (or poll)

the specified systems at the
designated hours. (See Poll).

uudemon.day                                     UUCP maintenance tasks to be
run once per day.

uudemon.kr                                      Run uucico once per hour and
process all pending work.

uudemon.wk                                      UUCP maintenance tasks to be
performed once per week.
uugetty                 Alternate getty program.
uusched                 The uucp scheduler.
uutry                   Debugging program.

uusub                                           uusub defines a UUCP subnetwork
and monitors the traffic

between the hosts.

uuxqt                   uuxqt                   Job execution on our
system.

SEQF                                            The sequence number for the
next job.

L_stat                                          System status file.

L_sub                                           UUCP connection statistics.

R_stat                                          Request status file.

R_sub                                           UUCP traffic statistics.
