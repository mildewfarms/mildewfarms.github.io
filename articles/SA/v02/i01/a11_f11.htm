Figure 10: The uupick command

uupick [options] file

options
-sSYS           Pick up the files from the remote system SYS.

If the -s option is not used, all of the files sent using uuto will be
available for pick up.

Interactive Commands

The prompt from uupick is "?"

*               List the interactive commands.

d               Delete the command without copying it.

m [dir]         Move the file into the current or
specified directory.

a [dir]         Move _all_ of the files from the same
system into the current or the specified
directory.

p               Print the contents of the file on the
terminal screen.

q or Ctrl-d     Quit.

!cmd            Escape to the shell and run the specified
command.
