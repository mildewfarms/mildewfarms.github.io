Figure 2: Devices file

Name/Type       Device  Dialer  Speed   dialer-token

INTACU          tty01   -               1200    intacu

This entry defines a device named as INTACU, which is accessed through
/dev/tty01, using the intacu dialer program, at 1200 baud only.

TBIT            tty17   -               Any     TBIT

This entry defines a device named TBIT, which is accessed through
/dev/tty17, using the TBIT dialer program, and it will allow any
connect speed requested in the Systems file.

bugs            tty21   -               9600    direct

This entry defines a device named bugs, which is the name of one of the
machines in my local UUCP network.  The dialer program is direct,
indicating that this is a direct connection, and connections are done
at 9600 baud only.

TCP             TCP,e   -               Any     TCP             540

This defines a device used for TCP/IP connections using the "e"
protocol.  There is no dialer port, and Any speed can be used.  The TCP
in the dialer field indicates that this is a TCP connection to be made on
the port specified in the token field, TCP port 540.
