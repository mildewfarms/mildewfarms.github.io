/usr/bin
This contains the user commands, such as
cu              * interactive dialup access
uucp            * job scheduler
uudecode        * decode files
uuencode        * encode files
uulog           * show UUCP log files
uuname          * get UUCP names
uupick          * pick files
uustat          * get UUCP status
uuto            * send files
uux             * command execution
