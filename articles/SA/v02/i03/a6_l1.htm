Listing 1: "bad.netrc" template

Subject:Security Notice
Text:
An automated system security check has detected that
your .netrc file has a protection mode that allows
anyone to read it.  Since this file often contains
readable passwords, this may not be a good idea.

I STRONGLY suggest that you give the command

chmod go-r ~/.netrc

at your next opportunity. Please let me know if you
have any questions or problems.
