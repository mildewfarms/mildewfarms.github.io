Listing 1

Sample shell script to automate the process of getting the file for edit, running vi, and prompting for a delta. The only argument required is the name of the file to edit. Do not use the s. prefix on the file name. 

1       :
2       if [ ! "$1" ]
3       then
4       echo "Usage : $0 file"
5       exit 1
6       fi
7       echo "extracting $1 ..."
8       get -e s.$1
9       vi $1
10      echo "Enter a description of the changes made."
11      delta s.$1
12      exit 0
