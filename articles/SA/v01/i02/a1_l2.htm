Listing 2

Here are some sample a L.sys or Systems file lines suitable for UUNET's 900 number:

# Simple line.
uunet Any ACU 19200 1-900-468-7727 in:--in:--in: uucp
#
# Set up for a Telebit.                                      CONNECT "" login: uucp
uunet Any cua0 19200 cua0 "" ATX0S50=255S111=30DT19004687727
All modems on the 900 lines are Telebit T2500s.  Note that these modems first answer
with V.32, then at 2400, 1200, and last with PEP tones. This "900" number
charges $.50US per minute to the caller.
