Sidebar: Obtaining the Programs

Elm is freely distributable and available from any comp.sources.unix archive site. The current version, 2.3 Patchlevel 11 (2.3PL11), was released as Volume 22, Issues 60-85, with the 11 patches being Volume 24, Issues 25-35. In addition, Elm is available from several FTP sites around the world, and each of the Elm FTP sites has a file consisting of the pre-patched 2.3PL11 source. For a current list of FTP sites, send the message

send elm elm.ftp

to archive-server@dsi.com (or dsinc!archive-server). Elm is also available via anonymous UUCP from dsinc. For information on retrieving Elm via anonymous UUCP, send the message:



send anon how-to dir

to archive-server@dsi.com (or dsinc!archive-server). Note that the archive-server is a program, not a person, so send only those lines in the message.

Metaconfig, a portion of the Dist 2.0 package, was released by Larry Wall as comp.sources.unix Volume 16, Issues 34-42. It requires Perl version 3.0 or later to run. For Perl 4.0 Larry rewrote Metaconfig and plans are currently being discussed for a new release of the Dist package. Both Perl and Dist are covered by the GNU General License and are freely distributable.
