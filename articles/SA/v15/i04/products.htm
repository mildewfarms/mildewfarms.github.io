New Products

Asigra Releases Televaulting 6.0

Asigra announced Asigra Televaulting 6.0, which includes upgrades and new capabilities such as Oracle 10g and SQL 8.0 support, centralized mass deployment, and cross-platform protection for Unix, Linux, Windows, and Mac environments.

According to the company, Televaulting 6.0 provides hot agentless database backup for DB2, Oracle 10g, and SQL 8.0, which allows database users to operate while backup operations are in process without disruption. The software also includes a block-level resume feature for Oracle so that if a connection is lost, the backup session can be continued at the block level rather than the file level.

For more information, visit: http://www.asigra.com.

Aventail Launches Mac and Linux SSL VPN Solution

Aventail announced an SSL VPN solution for Mac and Linux platforms. According to the company, Aventail Connect Tunnel for Mac and Linux provides a complete in-office experience for end users and centralized management for network administrators managing a mixed-platform environment. Aventail Connect Tunnel delivers the same experience on all devices and platforms including Mac, Linux, Windows, and Windows Mobile.

Mac and Linux users have two secure, remote access options: Connect Tunnel, a light-weight, persistent agent for complete network application access; and WorkPlace Access, clientless browser access for Web applications and file share access. Support for Mac and Linux browsers including Safari, Firefox, Mozilla, and Microsoft Internet Explorer. For more information, visit: http://www.aventail.com.

Paragon Software Group Launches NTFS for Linux 5.0

Paragon Software Group introduced a solution that allows Windows and Linux communicate. According to the company, NTFS for Linux Standard Solution is available in a Personal Edition, which includes the fully functional NTFS for Linux driver with maximum performance and stability and also offers a recovery bootable CD with the build-in driver; and Professional Edition that offers additional utilities such as the ability to check integrity, create NTFS volumes, perform NTFS file system-related tasks, and copy (backup/restore) files/directories.

For more information, visit: http://www.paragon-gmbh.com.

nCipher Unveils KeepSecure Product Suite

nCipher plc announced the nCipher KeepSecure product suite. According to the company, KeepSecure is an enterprise-wide data protection capable of encrypting data at the database, file, and application levels. The KeepSecure suite uses encryption and access control services delivered as a series of security adapters that have been specifically designed for integration with popular databases, file systems, and specific business applications.

For more information, visit: http://www.ncipher.com.

Boundless Corporation Announces Linux Web Terminal Product Lines

Boundless Corporation announced additions to its line of Linux Web Terminals. According to the company, the ADDS 5600 Terminal is a replacement for the ADDS 5500 series Web Terminal. The ADDS 5700 is a high-resolution (1600 x 1200 DVI) terminal with Dual Video (VGA and DVI) capabilities. Both the ADDS 5600 and 5700 terminals feature a 1.0Ghz Via C3 processor fan-less design, expandable memory, the Mozilla Firefox browser, and an optional Smart Card Reader.

For more information, visit: http://www.boundless.com.

ServerPronto Introduces the Gigabit Port

ServerPronto recently announced the gigabit port, which allows customers to absorb malicious attacks, or massive intended traffic spikes, without adversely affecting mission-critical applications. According to the company, ServerPronto's gigabit port option is available to customers for monthly fee.

Unlike shared hosting, where multiple sites are hosted on a single server, dedicated hosting allows companies to rent the entire server giving them 100 percent of the system's resources. Beyond multiple port options and affordability, ServerPronto offers many other customizable features, Linux options, power, server-grade parts, and a 99.999 percent uptime guarantee.

For more information, visit: http://www.serverpronto.com/.

Novell announced it will bundle support and training offerings with SUSE Linux Enterprise Server. This offering is available to new customers, effective immediately. For more information, visit: http://www.novell.com/linuxserverandservices.
