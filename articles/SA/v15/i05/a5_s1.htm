Java Application Hack

If you have ever worked with Java on the Unix platform, you may have noticed that in the process table, the Java application shows only the Java binary followed by the arguments.

A fix that works without breaking the application is to pass -D, which sets a system property. By passing the -D flag, you can define a generic parameter that the JVM will discard.
