Welcome

Welcome to Version 13 of the Sys Admin CD-ROM. This version includes all of the articles and associated code from the beginning of Sys Admin magazine in 1992 through August of 2007. That includes more than 100 issues' worth of articles from your resource for systems administration solutions. We've also included some Web-exclusive articles not published in the magazine.

This CD also includes all issues of The Perl Journal from 1996 through 2002. Both the Sys Admin and TPJ articles are presented in HTML and ASCII format, along with all source source code and related files. This CD-ROM is browsable with any frames-capable, Java-enabled Web browser on any platform.

We are not able to provide technical support for the code included here, but we encourage you to use the information in our articles and code provided by our hard-working contributors.
