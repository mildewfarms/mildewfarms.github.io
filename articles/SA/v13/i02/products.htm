New Products

Xandros Announces Xandros Desktop 2

Xandros, Inc. recently announced the availability of version 2 of the Xandros Desktop OS. According to the company, Xandros Desktop 2 offers an intuitive, graphical environment that installs with four clicks of a mouse. Xandros Desktop 2 provides integration with Microsoft Windows programs and networks, plus new usability enhancements such as drag-and-drop CD burning in Xandros File Manager. The new release employs a Xandros-enhanced KDE 3.1.4 and an underlying 2.4.22 Linux kernel. For more information, visit: http://www.xandros.com/products.html.

VanDyke Updates SecureCRT and SecureFX

VanDyke Software announced the release of version 4.1 of its SecureCRT secure remote access client and version 2.2 of its SecureFX multi-protocol file transfer client. According to the company, these releases offer increased integration between the client applications and advanced Kerberos authentication when connecting to a GSSAPI-enabled server like VanDyke's VShell for Windows and Unix. Using Kerberos authentication provides a single sign-on to the entire network that can now include secure remote access, secure file transfer, and secure data tunneling. The two applications now use a shared host key database -- once a server's credentials have been accepted by one application, the other also recognizes it as a known host, which can reduce end-user confusion about accepting host keys and reduce support costs. The private key agent cache is also shared, eliminating the need to reenter a passphrase once a connection has been established by either application when using public-key authentication.

Fully functional evaluation copies of SecureCRT, SecureFX, and VShell can be downloaded from the company's Web site at: http://www.vandyke.com/download/index.html.

Ecora Patch Manager 3.0 Adds Multi-Platform Support

Ecora recently announced the addition of Sun Solaris support in Patch Manager 3.0, its automated patch analysis and remediation software. According to the company, Ecora Patch Manager 3.0 helps IT staff to analyze for missing and installed patches, research, test, and deploy the patches across the enterprise to resolve both Windows and Solaris security vulnerabilities. Patch Manager 3.0 also includes new features such as Policy Manager for enforcing corporate patch policies, a browser-based Reporting Center, and Patch Test Center to assist lab-testing patches. For more information, visit: http://www.ecora.com.

SUSE LINUX and IBM Launch Software Integration Center

SUSE LINUX and IBM announced the foundation of a joint Software Integration Center located at the IBM Toronto Lab. According to the companies, the collaboration will initially focus on IBM DB2 Universal Database offerings and will subsequently be extended to other IBM software products. In the Software Integration Center, IBM and SUSE will collaborate on porting, migration, certification, and support of future IBM DB2 software offerings on the SUSE LINUX Enterprise Server platform. This will help ensure optimized functionality of database applications and underlying enterprise operating system technology, making SUSE LINUX Enterprise Server and IBM DB2 database software a highly available, scalable combination for mission-critical deployments in data centers. For more information, visit: http://www.suse.com.

Raritan Introduces Dominion SX

Raritan Computer, Inc. introduced the Dominion SX, a Web-based serial console server. According to the company, the Dominion SX helps enable IT professionals securely monitor, troubleshoot, configure, and manage all serial equipment in the enterprise -- using a LAN, WAN or Internet connection -- from anywhere, anytime. The Dominion SX -- a Web-based serial console server -- can manage any equipment that uses RS232 (serial or asynchronous) port connections, including servers running Linux, Solaris, Unix, and Windows 2003 Server; routers; hubs; switches; and PBXs. For more information, visit: http://www.raritan.com.

Command Prompt Unveils Mammoth PostgreSQL Replicator

Command Prompt announced the availability of its Mammoth PostgreSQL Replicator, an asynchronous replication system designed to be WAN and faulty connection tolerant. According to the company, Mammoth PostgreSQL is a robust, reliable, SQL-compatible Object Relational Database Management System (ORDBMS). For more information, visit: http://www.commandprompt.com.

Etnus Releases TotalView 6.3.1.

Etnus recently announced TotalView 6.3.1. According to the company, the new release includes support for Linux on AMD's Opteron computing systems, HP-UX on Intel Itanium-based systems, and the Absoft Fortran 95 version 8.2 compiler. TotalView also now supports Red Hat 9.0, Advanced Server 3.0, and SuSE Enterprise Server 8. For more information, visit: http://www.etnus.com.
