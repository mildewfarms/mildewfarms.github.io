TeraCloud Announces io_profile v1.0

TeraCloud Corporation announced the availability of a free IO profiling utility, io_profile. According to the company, io_profile is a free and open-source tool that measures and monitors the performance characteristics of a given IO stack. The portable command-line utility actively measures the I/O stack in Windows and Unix environments from an applications perspective. io_profile measures the performance of a large number of open-seek-read/write-close cycles at random locations in a set of randomly selected files. io_profile can help IT professionals conduct comparative analysis and tune IO channels for I/O size characteristics.

For more information, visit: http://www.tcloud.com/utilities.html

Globalstor Announces ExtremeStor-iNAS Server

Globalstor Data debuted the ExtremeStor-iNAS, NAS/iSCSI server. According to the company, ExtremeStor-iNAS incorporates as many as 24, 250GB hot-swappable SATA (Serial Advanced Technology Attachment) hard drives, and provides IT managers with up to 6TB of scaleable storage and low-cost, high-performance iSCSI file access in a single, consolidated centralized storage management solution. Supporting RAID levels 0, 1, 5 and 0+1, ExtremeStor-iNAS is an option for data-intensive enterprises offering storage pools for both workgroup and departmental configurations.

ExtremeStor-iNAS includes:

  • Linux Minimized OS
  • Web interface control, 128bit SSL, Mini SSL Web server
  • Error reporting -- Email notification, SNMP
  • Network Protocol Support -- CIFS/SMB, NFS, AppleTalk, network trunking, load balancing, failover, HTTP, FTP
  • XFS Journaling File System
  • Access Control List
  • Windows Active directory and Unix NIS Support
  • SNMP, NDMP
  • Dynamic volume resizing
  • SnapShot
  • Storage virtualization
  • Clustering/failover support
  • SCSI allows both Block I/O and File I/O
  • IP-Storage and IP-SAN
  • OS mirroring with automatic failover support

For more information, visit: http://www.globalstor.com

Open Country Releases Linux Software Management Tool

Open Country launched its system management solution for Linux. According to the company, the product is designed to help small departments with as few as five Linux systems, and enterprises with up to 5,000 Linux installations, reduce the costs, time, and complexity of managing Linux IT operations.

The company says that its software management tool lowers the TCO of running Linux systems and provides key system management functions. OC-Manager performs bare-metal operating system provisioning, software and patch management, hardware and software asset management, backup and restore, remote systems administration, and remote desktop takeover functions.

Priced from $9 per system per month, OC-Manager manages RPM-based Linux distributions including SuSE, Sun JDS, Red Hat, Fedora, Mandrake, and even legacy software from these same vendors, all running on the same network, according to the product's specifications. Open Country says its products adapt to an organization's existing infrastructure and offer companies a strategy to manage Linux without having to worry about retraining or investing in new IT administrators.

HP Announced StorageWorks Scalable File Share

HP announced a file sharing product that uses new Linux clustering technology to deliver up to 100 times more bandwidth than typical clusters. According to the company, HP StorageWorks Scalable File Share (HP SFS) is a self-contained file server that enables bandwidth to be shared by distributing files in parallel across clusters of industry-standard server and storage components. The product is the second based on HP's "storage grid" architecture and the first commercial research product to use a new Linux clustering technology, called Lustre, which was developed through collaboration between HP, the U.S. Department of Energy (DoE), and Cluster File Systems, Inc. HP SFS allows applications to see a single file system image regardless of the number of servers or storage devices connected to it. Built using industry-standard HP ProLiant servers and HP StorageWorks disk arrays, HP SFS provides protection from hardware failures through resilient, redundant hardware and built-in failover and recovery.

For more information, visit: http://www.hp.com
