New Products

Innovation Data Processing Announces Backup Server

Innovation Data Processing released an addition to their FDR/UPSTREAM Storage Management family of products. According to the company, the UPSTREAM Reservoir backup server allows organizations to utilize UPSTREAM either in mixed z/OS mainframe, open systems environments, or entirely non-mainframe environments. The UPSTREAM Reservoir supports a variety of disk, tape, and robotic devices. Backup devices can be a single tape drive, tape library, silo, or disk arrays. The UPSTREAM Reservoir backup server can use any tape drive or library, provides the ability to back up to disk or tape, and provides the option for data to be backed up to disk, staged from disk and automatically moved to tape.

The UPSTREAM Reservoir has multi-platform support, including agents for Intel Linux and zLINUX, and also supports Windows NT/2000/XP/Server 2003, Novell, OS/2, AIX, Sun Solaris, HP-UX, and Tru64. For more information, visit: http://www.innovationdp.fdr.com.

3ware Introduces Multi-lane Controllers

3ware, Inc. announced new controllers for Serial ATA RAID systems. According to the company, the 3ware Escalade 8506 Series Multi-lane Internal (MI) controllers increase reliability and simplify installation by reducing the number of required connection cables. The 3ware Multi-lane controllers are designed with the same hardware, software, and firmware architecture of the 3ware Escalade 8506 SATA RAID controllers. The Escalade Multi-lane boards feature an integrated connector system that combines four SATA ports into a single connection on the controller side and a single connector on a backplane. For more information, visit: http://www.3ware.com.

Katharion Introduces ISP Anti-Spam Solution

Katharion recently announced an anti-spam solution for ISPs. According to the company, Katharion offers highly effective anti-spam and anti-virus defense at the message gateway, with options for handling of detected spam. Katharion's Anti-Spam Solution for ISPs deploys as a perimeter solution at the messaging gateway, requiring no software installation, integration, or ongoing maintenance by the ISP or by the ISP's customers. The solution also allows ISPs to manage multiple customers via straightforward Web-based administrative tools, and allows ISPs to offload administration to customers. For more information, visit: http://www.katharion.com.

Aspen Systems Unveils SuperServer 6113L-8

Aspen Systems introduced a high-end server solution in a 1U form factor that is optimized for HPC, large databases, clusters, and storage. According to the company, the SuperServer 6113L-8 is based on the E8870 chipset and has been optimized for dual Itanium 2 processors up to 1.5GHz with 6MB L3 cache (6.4GB/s system bus speed) in a 1U rackmount form factor. It features one P64H2 bridge controller and support for one 64-bit-133 MHz PCI-X slot and 4 SCSI or SATA drives in a 1U configuration. The 2U system will support up to three 64-bit-133 MHz PCI-X slots and 6 SCSI or SATA drives. The serverboard also offers support for up to 16GB of regular DDR SDRAM in eight DIMM slots, and dual Gigabit Ethernet ports for networking capabilities. One scalable node controller and four memory repeater hubs are also included. For more information about the SuperServer 6113L-8, visit: http://www.supermicro.com/

Panda Software Updates Antiviruses for Linux Mail Servers

Panda Software released new versions of its antiviruses for Linux mail platforms (Sendmail, Qmail, and Postfix). According to the company, Panda SendmailSecure, QmailSecure, and PostfixSecure include new features that make them more powerful and effective, without affecting the performance of Linux servers. The new versions also include the option to scan fragmented messages.

Panda SendmailSecure, QmailSecure, and PostfixSecure can now detect whether a virus uses the iFRAME/MIME vulnerability exploit, which allows a new malicious code to trick the system using a false MIME header in the message. This feature blocks viruses that can run automatically when the message carrying it is viewed in the preview pane. These products can be managed through Panda Software's new administration console, Panda AdminSecure, which centralizes installation, configuration, and administration of other tasks related to these antivirus solutions in a single computer. Panda SendmailSecure, QmailSecure, and PostfixSecure can be purchased separately or as part of Panda EntepriSecure, Panda Software's security suite for corporate networks. For more information, visit: http://www.pandasoftware.com.

Cyclades Releases AlterPath PM10 and PM20

Cyclades Corporation introduced the AlterPath PM10 and PM20, which enable remote power control of servers and network gear. According to the company, the new products have the industry's highest density for IPDUs for both horizontal and vertical models. When used in conjunction with Cyclades console servers, the AlterPath PM10 and PM20 can deliver management capabilities and faster problem solving by integrating console access and power control into a single interface. IT professionals can remotely power off, power on, and reboot network devices using the same console management session from any location via telnet, Secure Shell (SSHv2), or through a secure Web session, which can help automate data center operations. The products, which support 120V with 15, 20, or 30 amps, can also provide accurate power measurement of each IPDU so temperature or current thresholds can be monitored. For more information, visit: http://www.cyclades.com.

Tidal Software Unveils Tidal Transporter

Tidal Software announced Tidal Transporter, a tool to automatically migrate Tidal Enterprise Scheduler job automation definitions from one master database to another. According to the company, Tidal Transporter does not require the user to stop his business production processing environment when migrating job schedule definitions from test environments. Transporter can migrate most relevant information (such as Job Definitions, Job Groups, Job Classes, Calendars, Events, Actions, and Variables) without impacting system availability and with fewer errors. For more information, visit: http://www.tidalwavesoftware.com.

For more information, visit: http://www.xandros.com.

Aduva Releases OnStage 2.1

Aduva announced the release of OnStage 2.1. According to the company, this release means companies do not have to be tied to a single Linux operating system (e.g., Red Hat or SuSE) or hardware platform (Intel or Mainframe). Extracting and retaining Linux system logic from installed Linux OS and hardware platforms, OnStage 2.1 can resolve operating system and hardware platform conflicts to improve system function. OnStage 2.1 can reduce the time and level of expertise needed for managing an up-to-date, secure, and reliable Linux software environment. It provides a real-time audit trail of the time and date of every change, installation, and uninstall on every host, by every user. For more information, visit: http://www.aduva.com.

CyberGuard Enhances SME575

CyberGuard Corporation announced the enhanced SME575, a SnapGear by CyberGuard VPN/firewall appliance with new intrusion detection system (IDS) features, plus advanced Web proxy cache technology that improves network security and performance. According to the company, the new IDS is built around Snort, open source software for intrusion detection. Snort can identify thousands of different attacks through a comprehensive signature database. Network administrators can use graphical open source tools to analyze alerts captured in the system logs to locate and correct weaknesses in perimeter defenses. With the new Web cache proxy features powered by Squid (open source software for proxy cache), the upgraded SME575 can also help customers add more users to an existing Internet connection without sacrificing performance. For more information, visit: http:// www.snapgear.com.
