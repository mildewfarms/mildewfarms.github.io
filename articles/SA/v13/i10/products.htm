New Products

F-Secure Anti-Virus for Linux Embedded in Meridius Security Gateway

F-Secure Corporation announced that BlueCat Networks, Inc. has licensed F-Secure Anti-Virus for Linux to embed into their Meridius Security Gateway appliance. According to the company, the Meridius Security Gateway, a 1U rack-mountable appliance powered by a hardened Linux kernel, is a secure solution for organizations battling spam and viruses.

Antivirus news is available at F-Secure's weblog at: http://www.f-secure.com/weblog/weblog.rdf. For more information, visit: http://www.F-Secure.com.

Globalstor Expands Scaleable Capacity of New NAS/iSCSI Servers to 9.6TB

Globalstor Data announced the availability of the ExtremeStor-iNAS400, an expanded version of their line of NAS/iSCSI servers. According to the company, ExtremeStor-iNAS400 incorporates as many as 24, 400GB hot-swappable SATA (Serial Advanced Technology Attachment) hard drives, and provides IT managers with up to 9.6TB of scaleable fault-tolerant storage and low-cost, high-performance iSCSI file access in a single, consolidated centralized storage management solution. Integrating Dual Intel Xeon 2.4Ghz processors, the ExtremeStor-iNAS400 can help ensure maximum computing power over a dual gigabit network interface for more efficient file retrieval. The ExtremeStor product line incorporates a Linux minimized OS on a solid state flash disk and comes standard with Ultra 320 SCSI. Organizations also have the option to add Fibre Channel HBA.

For more information, visit: http://www.globalstor.com.

Pogo Linux Unveils New Products

Pogo Linux Inc. recently announced an AMD64 Linux notebook and a turnkey Beowulf clustering solution. According to the company, the Pogo Linux KonaBook 3100 notebook is one of the first notebooks built from the ground up for Linux. PogoLinux also announced a new cluster in-a-box offering that gives customers high performance clustering capability at prices starting at less than $28,000 for 16 nodes. The Pogo Linux ComputeColony Beowulf Cluster Series is available in 16, 31, and 64-node configurations.

For more information, visit: http://www.pogolinux.com.

IPLocks Introduces IPLocks 4.1

IPLocks announced the availability of IPLocks version 4.1. According to the company, IPLocks 4.1 expands the functionality of its external, non-intrusive, cross-platform, highly scalable database security management system with the introduction of audit analysis for analyzing transactions. IPLocks 4.1 automates the detection and notification of database reads and changes and archives events to ensure data security, integrity, and availability for regulatory compliance. IPLocks 4.1 provides database monitoring, assessment, and real-time notification and facilitates regulatory compliance for major database platforms including Oracle, DB2, SQL Server, Sybase, and HiRDB (Hitachi). IPLocks 4.1 captures, displays, and archives data changes.

IPLocks 4.1 runs on any Pentium-based system with Linux OS or Windows OS. For more information, visit: http://www.iplocks.com.

Rackable Systems, Inc. announced the launch of the Scale Out Server Series, its server solution designed for large, high-density, data center deployments. According to the company, the Scale Out Server Series combines the key features of Rackable Systems' open architecture, rack-mount server design with improved density, serviceability, and management.

For more information, visit: http://www.rackable.com.

SSP-Litronic Introduces NetSign Privacy Agent Version 4.0

SSP Solutions, Inc. announced that the company is addressing enterprise challenges in privacy control and regulatory compliance with NetSign Privacy Agent (PA) Version 4.0, an integrated software suite for the enterprise desktop. According to the company, NetSign PA provides a solution for enterprises that require high assurance processes for securing and archiving electronic communications and digital files. NetSign PA 4.0 builds upon SSP-Litronic's Public Key Infrastructure (PKI) and smart card technologies to ensure the security and privacy of email and instant messaging communications. NetSign PA 4.0 utilizes digital certificates and PKI technology to verify user identities and certify the origin of email.

For more information, visit: http://www.ssliptronic.com.
