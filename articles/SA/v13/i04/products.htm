New Products

HELIOS Software Ships WebShare

HELIOS Software recently announced HELIOS WebShare. According to the company, HELIOS WebShare is a secure server solution that enables users to access their file servers over the Internet. WebShare can replace insecure file transfer methods such as FTP, VPN, ISDN, email, and others, using an intuitive Web browser interface to enable real-time file uploads and downloads, document proofing, and remote file management. In addition to remote file browsing, management, and transfer, HELIOS WebShare works with HELIOS ImageServer and HELIOS PDF HandShake to provide remote previews of PDF, QuarkXPress, and InDesign documents, as well as other major image formats.

For more information, visit: http://www.helios.de.

Guardian Digital Launches EnGarde Secure Linux

Guardian Digital, Inc. recently announced EnGarde Secure Linux. According to the company, EnGarde delivers features designed to ease the process of building a complete Internet presence and the level of security necessary to prevent system compromises. EnGarde features secure Web-based management of all functions, including Internet edge services (Web, DNS, email), integrated intrusion detection, cryptography, improved authentication and access control, as well as protection from many forms of intrusion such as buffer overruns and denial of service attacks. EnGarde Secure Linux features the Guardian Digital WebTool, a Web-based interface used to securely manage network and server operations. EnGarde Secure Linux is available for download from http://engardelinux.org and includes a free 30-day subscription to the Guardian Digital Secure Network to obtain system and security updates.

For more information, visit: http://www.guardiandigital.com.

Aptitune Releases SarCheck 6.0 for AIX

Aptitune announced the release of SarCheck Version 6.0 for AIX. According to the company, SarCheck is a performance management tool designed to assist systems administrators in the analysis of system performance by translating the output of several monitoring utilities into a plain English or HTML-formatted report. SarCheck identifies performance bottlenecks, finds runaway processes and memory leaks, recommends changes to various tunable parameters and hardware configurations, and quantifies remaining system capacity. SarCheck runs on many Unix platforms, and support for other languages is planned.

For more information, visit: http://www.sarcheck.com.

Procom Unveils DataFORCE 800R

Procom Technology recently added an enterprise-class model to its DataFORCE line of CD/DVD-ROM appliances that combines high-capacity storage and accelerated performance to store, organize, protect, and speed access to data based on optical media. According to the company, the new DataFORCE 800R provides up to 2 terabytes of RAID-5 protected storage and data access 220 times faster than optical media.

Supporting Windows, Linux, Macintosh, Unix, and Novell NetWare, the DataFORCE 800R can provide seamless integration into heterogeneous network environments. It connects all users on the network (via 10/100/1000 BaseT Ethernet), allowing cross-platform access and file sharing of thousands of CDs or hundreds of DVDs.

For more information, visit: http://www.procom.com.

Proteligent Introduces BirdsEye

Proteligent, Inc. announced BirdsEye, Web-based system-monitoring software to help prevent system downtime. According to the company, the new software tool was developed to support the IT infrastructure needs of small- and medium-sized businesses and address system downtime. BirdsEye provides systems administrators with information about servers, workstations, and network devices and their resources, even at remote locations.

Corrent Releases SR500 & SR600

Corrent Corporation introduced the Corrent SR500 and Corrent SR600, two integrated firewall and VPN security platforms. According to the company, the new line of enterprise-class integrated security appliances use Check Point SecurePlatform software and are based on IBM server platforms and are able to stop 1 Gbps of malicious traffic from an ongoing DDoS attack while at the same time allowing more than 948 Mbps of valid network traffic to pass. The Corrent SR500 and SR600 offer a connection rate of 50,000 sessions per second, with up to 1,000,000 concurrent connections and near real-time connection agility.

For more information, visit http://www.corrent.com.
