Important DNS Files

Resolver Configuration: The /etc/resolv.conf is used by most versions of UNIX to identify the client configuration of DNS, including name servers and domain names.

Solaris Name Server Activation: The /etc/nsswitch.conf file is used to tell Solaris how to resolve names -- either hosts, DNS, or NIS.

Zone files: BIND can be compiled with specific location of zone files, but the standard is /var/named/.
