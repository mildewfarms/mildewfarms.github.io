Alpine Step by Step

1. User enters IP or hostname into Alpine Web interface.

2. Alpine's zebra daemon determines target IP's most specific BGP prefix.

3. SNMP polling all routers learns every available BGP path for target prefix.

4. The Alpine server is configured with a separate IP address for each.

5. IPFW directs each source IP to correct egress router.

6. Router's filter-based forwarding matches source IP directing traffic out the correct ISP.

7. Alpine displays easy to understand diagnostic information.
