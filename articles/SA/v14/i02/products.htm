New Products

Texas Memory Systems Announces RamSan-120

Texas Memory Systems, Inc. announced availability of a new entry-level solid state disk, the RamSan-120. According to the company, the RamSan-120 is the fastest 1U solid state disk on the market and the first solid state disk ever to implement RAID-protected dual inline memory modules. The RamSan-120 delivers more than 70,000 random I/Os per second and up to 400 MB per second of bandwidth. Its 8 GB capacity is housed in a slim one rack-unit (1U) chassis.

The company also says that the RamSan-120 is as easy to install as an ordinary disk drive system. A single computer can use it with a host bus adapter, or multiple computers can access it on a Fibre Channel storage area network (SAN). Full monitoring and configuration capabilities are available over any browser via a protected Java applet. The RamSan-120 is fully SNMP compatible. For more information, visit: http://www.texmemsys.com

SANRAD Introduces iSCSI-Based Storage Solutions

SANRAD Incorporated announced the availability of a new family of intelligent iSCSI-based storage solutions. According to the company, the storage solutions feature SANRAD's V-Switch in conjunction high availability storage ranging from 500 GB to 5.4 PB (petabytes). Pricing for the family of total storage solutions starts at $15,000. The V-Switch 2000 combines the functionality of a multi-protocol switch, storage virtualization engine, storage router, and Fibre Channel gateway. The V-Switch 3000 blends an interoperable multiple-port iSCSI gateway with comprehensive StoragePro software to deliver cost-effective open, iSCSI connectivity across Ethernet environments. V-Switch 3000 also delivers fault tolerance and continuous data availability, network-based pooling and volume management, advanced traffic management, storage security, and remote management capabilities. For more information, visit: http://www.sanrad.com/

Radiant Data Unveils PeerFS v3.0

Radiant Data Corp. announced PeerFS Version 3.0, which, according to the company, is a peer-to-peer continuous data availability technology for Linux-based enterprise applications. PeerFS works semi-synchronously to provide seamless, consistent access to data. The data on all nodes is visible and usable by local applications. PeerFS enables simultaneous transactions on multiple servers in multiple locations with separate but identical data stores, which provides scaling capabilities for users running Linux on Apache, MySQL, PHP, and other Web services components.

Starting at $1,995, PeerFS 3.0 is sold through a network of resellers on a per-node basis. Channel partners are delivering several solution packages based on PeerFS, including blade and multi-processor servers, storage and other Linux software. A 30-day evaluation version of PeerFS v3.0 and more information are available at: http://www.radiantdata.com

Hendrickson Software Components Releases Cross-Platform SpamX

Hendrickson Software Components announced that SpamX, the company's software for spam source tracing and reporting, is available cross-platform for Windows, and now Macintosh and Linux. According to the company, SpamX allows users to effectively reduce spam in their email inboxes and on their networks. SpamX works by automatically tracing the source of spam and reporting offenders to their respective Internet Service Providers. SpamX users typically see their spam levels drop to near zero within four weeks.

SpamX is available in both client and server editions. The client edition is priced at $19.95, and the server edition is priced at $399.95. Fore more information, visit: http://www.hendricom.com

Alacos Announces Linux Migration Agent Professional

Alacos recently announced Linux Migration Agent Professional, software to reduce the cost of migrating desktop computers to Linux. According to the company, Linux Migration Agent Professional lets users move documents, email, and settings from a Windows PC to a Linux computer with the included data transfer cable. Linux Migration Agent Professional migrates email (Outlook, Outlook Express, and Netscape to Evolution), address books, folders, My Documents, IE favorites, IE home page, backgrounds, icons, documents, spreadsheets, photos, music, movies, graphics, databases, and more. For more information, visit: http://www.alacos.com
