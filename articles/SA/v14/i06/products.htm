New Products

Permeo Announces Base5

Permeo Technologies, Inc. announced Base5, an integrated software appliance delivering "zero-touch" SSL VPN, endpoint security services, and advanced information controls. According to the company, Base5 eliminates the administrative and integration complexity in delivering access and security services to managed and unmanaged devices. The result is a decrease in operational costs of more than 50%, based on total cost of ownership analysis.

Permeo's Base5 extends SSL VPN and endpoint security services to any user and virtually any application, without integrating multiple vendor solutions. It provides a unified policy enforcement and management framework that integrates SSL VPN, information control, browser security, malware protection, and host integrity checks. Base5's patent-pending session layer technology enables a "zero-touch" deployment model in which no remote client administration is required for the delivery of connectivity and endpoint security capabilities.

For more information, visit: http://www.permeo.com.

Raritan Computer Introduces CommandCenter 2.0 and Diminion KX

Raritan Computer Inc. introduced two new complementary enterprise-class products -- CommandCenter 2.0, a global management appliance, and the Dominion KX, a highly scalable digital KVM (keyboard/video/mouse) switch. According to the company, the products enable global companies to remotely and securely manage thousands of multi-platform servers and IT devices in their enterprises through an easy-to-use Web interface.

For more information, visit: http://www.raritan.com.

Sensory Networks Announces NodalCore-X SPU

Sensory Networks announced its next generation adaptive NodalCore-X Security Processor (SPU). According to the company, NodalCore-X is designed to offload and accelerate bottleneck operations involved with high performance network security applications including antivirus, antispam, intrusion detection/prevention, antispyware, network monitoring, and other content security applications.

NodalCore-X integrates into a range of network security equipment via plug and play PCI cards. The new NodalCore-X architecture has been improved to provide greater and more rapid extensibility with network downloadable custom hardware engines, coupled with a higher speed internal bus. Programming the NodalCore-X SPU is simplified through an SDK that contains a comprehensive set of APIs using industry standard paradigms, illustrative example code, and robust high-performance demonstration applications. NodalCore-X is available as a simple network-based upgrade for all currently shipping NodalCore-based products and will automatically be available to both existing and new customers in the next release of the NodalCore SDK.

For more information, visit: http://www.sensorynetworks.com.

BeInSync Unveils BeInSync Professional Version 1.5

BeInSync announced its new, professional version, secure peer-to-peer synchronization software. According to the company, BeInSync Pro 1.5 incorporates new feature enhancements that make it easier for individuals, mobile professionals, and small business users to synchronize, access, and share data between multiple computers. New features in version 1.5 are aimed at enhancing the user experience and simplifying the sharing of files with partners, colleagues, family, or friends.

BeInSync Pro Version 1.5 is immediately available for download on the BeInSync Web site for an introductory price of $59.95 per year, which allows the users to install the software on up to three of their computers. Users can download the software for a free 30-day trial. The free version of the software will remain available and will allow users to synchronize up to a fixed number of files per day. For more information, visit: http://www.beinsync.com.

ZyXEL Releases Palm-Sized Portable Personal Firewall

ZyXEL Communications introduced its ZyWALL P1, a palm-sized portable personal firewall. According to the company, the ZyWALL P1 is easy to install and manage centrally by administrators to enforce corporate security policies. With the functions of ZyWALL P1, the IT staff can reinforce end-point security on business travelers or telecommuters. ZyWALL P1 is a compact and portable device that boasts a WAN port and a LAN port, and can be powered by the USB port. The ZyWALL P1 is a ICSA-certified firewall with 90Mbps throughput, as well as comprehensive SPI, DoS/DDoS, and Access Control features. ZyWALL P1 also provides an ICSA-certified IPSec VPN tunnel with hardware-accelerated DES/3DES/AES encryption of up to 34Mbps throughput.

For more information, visit: http://www.zyxel.com.

ActiveGrid Releases Open Source Grid Application Server

ActiveGrid, Inc. announced the early-access release of the company's next-generation enterprise application platform. According to the company, the ActiveGrid Application Builder and Grid Application Server are now available for download under the open source Apache Software License 2.0. Built on the LAMP (Linux, Apache, MySQL, PHP/Python/Perl) software stack, the ActiveGrid platform is designed to scale business applications across horizontal grids of commodity computers -- accelerating composite application development, reducing cost, and enabling customization at runtime. Developers can access and install the product code by downloading the application at: http://www.activegrid.com/developergrid.
