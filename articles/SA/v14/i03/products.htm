New Products

Opsware Releases Opsware Asset Tracking Edition 5.5

Opsware Inc. announced a new release of Opsware Asset Tracking Edition (ATE) Version 5.5. According to the company, the new release extends automated discovery capabilities to additional server and desktop platforms; adds new reporting capabilities; and provides more stringent access controls for improved, centralized security. ATE version 5.5 adds 64-bit computing support as well as support for the latest versions of the Red Hat Linux platform. It continues to support Windows, Solaris, HP-UX, and AIX.

For more information, visit: http://www.opsware.com

CrossTec Introduces NetOp Desktop Firewall

CrossTec Corporation announced the release of the NetOp Desktop Firewall, developed by Danware Data A/S and designed to protect enterprises from the vulnerabilities associated with the use of laptop machines on a corporate network. According to the company, NetOp Desktop Firewall is a driver-centric firewall and addresses the security needs that corporations encounter when their employees take their corporate laptops outside the protection of the perimeter firewall. The NetOp Desktop Firewall offers process and application control, protecting computers on the corporate network and ensuring laptop security.

The centrally managed firewall restricts traffic in to and out of network machines via port, protocol, and IP address. Additionally, the NetOp Desktop Firewall provides process control to limit which applications run on each user's machine. Only applications significant to a user's needs are run on their computers, controlling the execution of malicious applications such as viruses, worms, spy-ware, and ad-ware. The firewall's centralized management utility enables the creation of corporate policies that provide protection whether on or off the corporate network.

For more information, visit: http://www.crossteccorp.com/netopfirewall/index.html

Oracle Releases Oracle Database Lite 10g

Oracle announced Oracle Database Lite 10g, a database that extends the power of grid computing to the mobile workforce. According to the company, the new database is an integrated solution for developing and deploying vital database applications for mobile environments. It delivers features common in mission-critical systems to mobile and embedded devices. It facilitates persistent access to critical information and applications without requiring continuous connectivity to back-end enterprise systems.

Oracle Database Lite 10g is available immediately for purchase for $100 per named user licensing or evaluation download under a development license from the Oracle Technology Network. The software is available on all major operating systems including Linux, Unix, and Windows.

For more information, visit: http://www.cxotoday.com/cxo/jsp/article.jsp?article_id=2592&cat_id=911

Vanquish Announces Clean eMail Program

Vanquish recently announced a filterless zero-spam email platform for computer resellers and IT service providers nationwide. According to the company, the Vanquish clean-email program allows providers to generate new revenues by reselling the Vanquish Anti-Spam Appliance or by using the Vanquish Virtual Service to host email accounts for customers. Vanquish does not rely on software filters that trap messages based on tripwire words and phrases. Vanquish guarantees full and unfettered delivery of email from trusted Web sites and trusted senders no matter what kind of words they might use in the message.

Entry level pricing begins with the Vanquish Virtual Service starting at $50.00 annually for 10 clean email accounts, ranging up to $1,800 annually for 500 email accounts. The Vanquish Anti-Spam Appliance ranges in price from $1,399 for 750 users to $4,999 for 15,000 users. Respective annual software updates range from $270.00 to $1,350.

For more information, visit: http://www.vanquish.com

CXL Unveils AZScan

CXL announced AZScan, a PC-based application providing cross-platform security reviews for Unix, OS/400, and OpenVMS systems. According to the company, CXL developed AZScan, an application for reviewing these systems quickly and without disruption to the live systems. Because CXL's AZScan runs on an independent, standalone PC, it cannot interfere with the mid-range systems. There is no software to load onto the system; copy two or three key files onto the Windows PC and AZScan produces the security reports in text, Word, or HTML format.

For more information, visit: http://www.cxlsecure.com

Vanguard Introduces Vanguard Security Solutions 5.3

Vanguard Integrity Professionals announced the general availability of its newest release of security software, Vanguard Security Solutions 5.3. According to the company, Vanguard Security Solutions provides customers with enterprise-wide security on demand in cross-platform environments. This newest release includes a new product, Vanguard ez/Token, a two-factor authentication solution that integrates token technology with RACF. Vanguard Security Solutions 5.3 offers more than 40 reporting, auditing, and efficiency enhancements designed to meet the regulatory requirements for compliance with Sarbanes-Oxley, the Gramm-Leach-Bliley Act, HIPAA, etc.

Vanguard security products include intrusion detection, user reset of forgotten passwords, single password enterprise sign on, and a graphical user interface to RACF. RACF is the IBM security management product for the zSeries operating system, providing access to users who have authorization to use a requested computer resource.

For more information, visit: http://www.go2vanguard.com
