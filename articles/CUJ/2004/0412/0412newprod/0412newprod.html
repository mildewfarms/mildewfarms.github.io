New Products

C/C++ Users Journal December, 2004

AidAim Software Offers IM SDK

MsgCommunicator is a C++Builder/Delphi SDK for custom client/server instant messaging applications with data compression and strong encryption. MsgCommunicator allows sending/receiving binary buffers, streams, and string messages, and offers three data-compression algorithms (ZLib, BZip, PPM) along with nine compression modes for each. MsgCommunicator also provides messenger functionality to work with ContactList, UserInfo, Users Database, and the like. Contact +1-530-420-3624 or http://www.aidaim.com/.

Scientific Component Launches GPS ToolKit.NET

Scientific Component has released GPS ToolKit.NET for Windows—a native .NET-managed component for adding GPS support to any Visual Basic .NET, C#, or C++ project. GPS ToolKit.NET gives you an API for reading real-time data from GPS units. It provides access to dozens of data sets, including position, speed, course, satellite, and navigation data, in a number of formats and units. Additionally, GPS ToolKit.NET raises events each time a new fix is received, or navigation or satellite data is updated. Contact +1-267-448-4929 or http://www.scientificcomponent.com/.

SoftIntegration Ch Updates

SoftIntegration is offering Ch 4.7, Ch SDK 4.7, and Embedded Ch 4.7. Ch is an embeddable C/C++ interpreter for cross-platform scripting, 2D/3D plotting, numerical computing, shell programming, and embedded scripting. The 4.7 version features improvements for embedded scripting, and the Ch shell is now more compatible with the C shell in both command mode and shell programming. In all, 36 new APIs have been added in Ch SDK and Embedded Ch for easier interface with binary C/C++ libraries and embedded scripting. Enhancements for Ch SDK and Embedded Ch include input/output redirection, obtaining complete information about variables and functions in a Ch script, and interfacing functions with a variable number of arguments. Contact +1-530-297-7398 or http://www.softintegration.com/.

Wavelink Supports RFID Development

Wavelink is incorporating Radio Frequency Identification (RFID) development tools into its application middleware products, Wavelink Studio and Wavelink Emulation. You can tightly integrate your applications with other workflow and data collection functions, such as barcode scanning, point-of-sale credit-card reading, and wireless printing. Wavelink also lets you extend applications on mainframes to interact with RFID tags through Wavelink's Emulators. Contact +1-425-823-0111 or http://www.wavelink.com/.

ITTIA Announces db.*

ITTIA is launching db.*, an open-source, small-footprint embedded database. db.* is a combination relational- and network-model database; for example, frequently accessed data, or any data for which speed is paramount, can be stored in network model and all other data can be stored in a relational model. The database supports multiple users through its lock manager and has multistage database commits. It is programmed through a C API. Contact +1-425-462-0046 or http://www.ittia.com/.

CSS and OGIS-RI Collaborate on BizZyme

BizZyme, a Service-Oriented Architecture (SOA) solution from Creative Science Systems (CSS) and The Osaka Gas Information System Research Institute (OGIS-RI), lets you create graphical designs of business process logic that automatically translate into executable web services with process persistence support. OGIS-RI contributed XGrotto (an enhanced registry for web services that runs under the Tomcat/Apache platform) and CSS provided NetZyme (integration middleware that supports C/C++ connectivity, enables pervasive computing with mobile wireless Internet devices and can be used to design and manage any .NET application). Contact +1-408-369-1555 or http://www.creativescience.com/.

PrismTech Offers SDR Development Suite

PrismTech has announced OpenFusion e*ORB SDR, the first member of a middleware family specifically designed for wireless OEMs. OpenFusion e*ORB SDR (short for Software Defined Radio) is a suite of SCA-compliant, low-footprint CORBA ORBs (and related Common Object Services) optimized for low-profile and resource-constrained environments. The suite offers an interoperable solution for GPPs, DSPs, and FPGAs (e.g., in a radio's signal processing subsystem). Contact +1-781-270-1177 or http://www.prismtechnologies.com/.

Metrowerks Releases CodeWarrior Development Studio for PSP

Metrowerks is offering a version of the CodeWarrior Development Studio for Sony's PSP handheld gaming system. Version 1.0 of the studio includes the Metrowerks compiler; support for Sony Computer Entertainment's communication protocol; the ability to work within either the native CodeWarrior development environment or the Visual Studio.NET environment; and a C/C++ debugger that provides a graphic interface for code visualization. Contact +1-512-996-5300 or http://www.metrowerks.com/.

Configurable Analog Modules Available from Anadigm

Anadigm is offering programmable analog building blocks for free download. Each of the Configurable Analog Modules (CAMs) in this new "Customer Collection" can be downloaded to an Anadigm Field Programmable Analog Array (FPAA) to implement a software-controlled function in an analog equivalent to the FPGA. The first Customer Collection CAMs include a sawtooth and square wave oscillator, a triangle and square wave oscillator, a sum/difference stage with biquadratic filter, a multiplier with low-corner frequency LPF, and a voltage-controlled sample and hold. Contact +1-408-879-6677 or http://www.anadigm.com/.
