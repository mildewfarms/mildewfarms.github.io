C/C++ Interpreter Ch 5.1 and Embedded Ch 5.1

SoftIntegration has announced the availability of Ch 5.1 and Embedded Ch 5.1. Ch is an embeddable C/C++ interpreter for cross-platform scripting, 2D/3D plotting, numerical computing, shell programming, and embedded scripting. This release enhances embedded Ch scripting with thread handling, allows you to abort an embedded Ch thread at runtime, and supports MingW for SDK and Embedded Ch SDK. Ch in Windows contains more than 100 new UNIX utility programs with over 200 utilities in total for cross-platform shell programming and rapid software development. Ch and Embedded Ch are available for Windows, Linux x86, Linux PPC, Mac OS X, Solaris, HP-UX, FreeBSD, and QNX Neutrino RTOS. Ch Standard Edition is free for both commercial and noncommercial uses in all platforms. Ch Professional Edition is free for academic use and costs $399 for personal or commercial use. Contact +1-530-297-7398 or http://www.softintegration.com/.

Bandicoot CodeClipper

Bandicoot Software has announced the release of Bandicoot CodeClipper V1.0, a program that stores and organizes code snippets and provides searching facilities to find commonly used algorithms quickly and easily. CodeClipper provides a Shortcuts bar that allows for drag-and-drop of commonly used code clips into any IDE. An export and import facility allows the sharing of code between programming teams. CodeClipper is available for Windows (any version) and Mac OS X (Panther and Tiger) operating systems. Contact +61-7-3398-6131 or http://www.bandicootsoftware.com.au/.

Anthill Pro 2.4

Urbancode has announced the release of Anthill Pro 2.4, the latest version of their Build Management Server (BMS) software. Anthill Pro automates the software build process. At preset intervals or scheduled times (or on-demand), Anthill Pro can check out code from a repository, launch software compilers and builders, apply versions and labels according to a preset scheme, publish the results and artifacts of the build, and notify all interested parties by e-mail of the results, all while maintaining the build history in a web-based interface. Version 2.4 improvements include new embedded database technology that increases reliability, scalability, and speed; cleaner, more attractive GUI elements; integration with BitKeeper source control software; and a new Subversion driver. Contact +1-216-858-9000, info@ urbancode.com or http://www.urbancode.com/products/anthillpro/.

Deskzilla 1.0

ALM Works has announced the release of Deskzilla 1.0, a desktop client for the Bugzilla bug tracking system. The program lets you view a project as a hierarchical structure with nested queries. Bug counters in Deskzilla give users insight into the state of the project. You can search by entering search terms in a single box, or you can create a complex filter using the Query Builder. Deskzilla may be used to update bugs and to submit new bugs. If it detects a bug update collision, which typically comes as a result of two people changing the same bug simultaneously, Deskzilla lets you carefully review both change sets and resolve a conflict instantly or put off conflict resolution for a later time. Deskzilla 1.0 runs under Windows 98/Me/2000/ XP/Server 2003, Linux, and Mac OS X. The client is compatible with Bugzilla Version 2.14 and newer. Contact +1-733-598-5805 or http://deskzilla.com/.
