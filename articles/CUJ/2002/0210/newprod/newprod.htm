Cypress Releases Linux USB 2.0

Cypress Semiconductor Corporation has announced Linux USB 2.0, its host driver supporting the SL811HS embedded host/peripheral controller. Linux USB 2.0 provides Linux developers with the ability to add USB host functionality to a variety of embedded-system applications, such as cell phones, MP3 players, network appliances, and PDAs.

Currently, the Linux 2.44 kernel supports the OHCI (Open Host Controller Interface) and the UHCI (Universal Host Controller Interface) specifications for USB host controllers, which are typically implemented in a PC architecture and environment. Since Cypress’s SL811HS is a memory-mapped device, the release of this new driver allows the Linux host stack to provide USB host support without realizing that there is a non-OHCI/UHCI type host controller.

For pricing and additional information, contact Cypress Semiconductor Corporation, +1.408.943.2600; <www.cypress.com>.

Software Structures Announces TransDbg v1.0

Software Structures, Inc., has released TransDbg v1.0, a MSDTC (Microsoft Distributed Transaction) trace debugger. TransDbg tracks MSDTC calls into resource managers participating in user-selected transactions. At run time, TransDbg updates a list box with a trace of commands issued by MSDTC to resource managers enlisted in those selected transactions. All four ITransactionResourceAsync callbacks are traced: .PrepareRequest(), .CommitRequest(), .AbortRequest(), and .TMDown(). In addition, TransDbg tests and views the response of your transactions network design to resource manager failures. TransDbg may be set to fail any one of the various transactions.

TransDbg v1.0 requires Windows 2000. TransDbg v1.0 costs $5.50 and can be downloaded from the company’s website.

For more information, contact Software Structures, Inc., +1.713.781.6633; FewDollarSoftware@SoftwareStructures.com; <www.SoftwareStructures.com>.
