AutomatedQA Debuts AQtime .Net Edition

AutomatedQA Corporation has debuted AQtime .Net Edition, a performance optimization toolset for Microsoft’s .NET framework and the Visual Studio .NET development environment.

AQtime .NET Edition has three static profilers that analyze the MSIL executable and provide detailed statistics. The .NET Static Analysis Profiler reports which routines exist in the application, from where they are called in source, and what they call in turn. This does not tell if and when the calls will execute, but it does give a full report of routine interdependence in the source. The .NET Static Class Analysis Profiler extracts class relationships (server-client) from method-call information. The .NET Sequence Diagram Link Profiler creates a UML-style diagram that graphically represents the sequence of method calls. Classes are shown as entities, and source-code calls between methods of different classes as messages.

AQtime .Net Edition costs $349 for a single-user license. Volume discounts are available.

For more information, contact AutomatedQA Corporation, +1.702.891.9424; sales@automatedqa.com; www.automatedqa.com.
