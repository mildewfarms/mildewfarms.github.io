Lineo Debuts Lineo Embedix BridgeWorks VX and PS

Lineo, Inc. has released Lineo Embedix BridgeWorks VX and PS, a real-time solution for porting legacy code from VxWorks or pSOS+ to embedded Linux. Lineo has created a new, clean room implementation of RTOS services that operates natively on the legacy API and runs ahead of Linux. This architecture provides such features as real-time services with microsecond performance by default. With this architecture, Linux kernel activities cannot impede the legacy API real-time layer. In addition, BridgeWorks provides a flat memory model, as used by the legacy RTOS.

Lineo provides the Embedix BridgeWorks VX and PS products (including a comprehensive migration guide) as optional packages for integration with the Lineo Embedix SDK and the associated Lineo board support package. Embedix BridgeWorks VX and PS add-ins are available for x86, PPC, and MIPS platforms. Each add-in costs $2,500 for the first seat and $1,500 for each additional seat. Source code for both products is available under an optional license.

Extended Systems Releases XTNDConnect Mobile Objects

Extended Systems has released XTNDConnect Mobile Objects, a mobile application SDK. Optimized for wireless networks, such as an 802.11b/WiFi, XTNDConnect Mobile Objects offers a simple three-step development process. With XTNDConnect Mobile Objects, first developers build a server-based COM object (or use an existing one). Next, they use the XTNDConnect mobile Objects Client Interface Expert to generate a native (non-COM) interface for the client application. Finally, they use the object interface generated in step 2 with AppForge or CodeWarrior to integrate the server object into the client application. Other features include Streamlined Security based on Certicom Elliptical Curve technology, state or stateless connection models, and live database access with store and forward architecture for unstable network environments.

XTNDConnect Mobile Objects costs $395 per SDK and $595 per server.

For more information, contact Extended Systems, 800.235.7576; www.extendedsystems.com.
