Distributed Systems Research Announces Jess 6.0b1

Distributed Systems Research has released Jess 6.0b1, a rule engine and scripting environment written entirely in Java by Ernest Friedman-Hill at Sandia National Laboratories. Jess was originally inspired by the CLIPS expert system shell, but has grown into a distinct Java-influenced environment of its own. Jess 6.0b1 is used to build Java applets and applications that have the capacity to “reason” using knowledge supplied in the form of declarative rules. Jess 6.0b1 supports modules and includes enhancements and bug fixes.

Jess 6.0b1 is free for academic and government use. Flexible commercial licensing is available. A free, 15-day trial version can be downloaded from the website.

For more information, please contact Distributed Systems Research, 800.472.6342 or +1.925.294.2154; herzberg.ca.sandia.gov.

Jungo Software Technologies Launches Driver Development Tools v5.03 and CompactPCI Hot Swap Software

Jungo Software Technologies, Inc. has released version 5.03 of its driver development tools: WinDriver, KernelDriver, and Hardware Debugger. WinDriver is a driver development toolkit that simplifies the creation of monolithic device drivers for custom hardware. KernelDriver is a driver development toolkit that simplifies the creation of kernel-mode drivers, especially Layered (filter and miniport) and Standard drivers. Hardware Debugger is a point-and-click application that allows detection of USB, PCI, CompactPCI, ISA, ISAPnP, and EISA hardware and their resources without having to develop a driver.

Jungo has also released its GO-HotSwap software, which adds all the necessary software modules required to support CompactPCI Hot Swap, such as Hot Swap events detection, dynamic resource allocation (while the system is running), and PCI registers initialization.

A free full-featured evaluation copy of each product can be downloaded from the company’s website.

Datalight Unveils FlashFX 5.0

Datalight has released FlashFX 5.0, flash media management software that simplifies the configuration of resident flash arrays in Microsoft Windows CE-based devices. FlashFX 5.0 presents flash memory as a disk drive to Windows CE and applications. With FlashFX 5.0, developers open, read, write, and delete files in the same way that applications interact with hard-disk and floppy drives. FlashFX 5.0 automatically manages the required hardware interactions. The Datalight seamless flash architecture found in FlashFX 5.0 makes it possible for FlashFX 5.0 to allow Windows CE designers to boot Windows CE from the flash device and also to save and restore the Windows CE registry in nonvolatile flash.

FlashFX 5.0 is provided as source code in the FlashFX 5.0 Porting Kit, available for $2,500.

For more information, please contact Datalight, www.datalight.com.
