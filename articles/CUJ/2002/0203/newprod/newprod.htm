Altium Limited Announces Tasking C166/ST10 Software Development Toolset v7.5

Altium Limited has released the Tasking C166/ST10 Software Development Toolset v7.5, which offers a new GSO (global storage optimization) technique. This feature of the C/C++/EC++ compilers maximizes memory usage by automatically filling in memory qualifiers based on the suitability of objects in either near or far memory spaces. The optimization is global in its application scope, meaning that the whole application is pre-compiled first to gather information about all static objects and then compiled again using the optimum memory configuration as determined by the GSO.

Tasking C166/ST10 Software Development Toolset v7.5 starts at $2,650 for the C compiler and CrossView Pro simulator package. C166/ST10 customers with an active maintenance contract may upgrade for free. A demo version of the toolset is available from the company’s website.

Connectix Announces Virtual PC for Windows 4.2

Connectix Corporation has released Virtual PC for Windows 4.2 in conjunction with the release of Windows XP. Virtual PC for Windows 4.2 allows machines running Windows 2000/NT/Me/XP to host combinations of Intel-based operating systems running concurrently, including Windows 95/2000/NT/ME and Linux. Siroyan Offers OneDSP

Siroyan Limited has launched its high-performance OneDSP scalable processor architecture. OneDSP combines scalable, high-performance DSP with general-purpose RISC control code in a true, single core. OneDSP delivers scalable DSP performance through the application of VLIW (Very Long Instruction Word) clustering techniques.

Siroyan’s fully licensable OneDSP architecture exploits instruction-level parallelism, providing up to 32 execution-unit clusters to be implemented by the licensee with either 32- or 64-bit data paths. OneDSP will scale from 400 MMACs, with only the master cluster deployed, through 25,600 MMACs, with 32 clusters, at 200MHz.

The OneDSP architecture is delivered as a synthesizable core in RTL and is supported in a standard EDA development environment. The first soft core derived from OneDSP is designated SRA328, which can be configured with up to 8 execution-unit clusters with 32-bit data-paths, and includes an integrated AMBATM bus, debug port, and DMA.

For pricing and additional information, contact Siroyan Limited, +44.0.118.9497028; info@siroyan.com; www.siroyan.com.
