by Jim Hyslop and Herb Sutter
Many people know how to partially specialize a template. But what happens when the base template has a template parameter with a default value? The answer may not be quite what Bob expected.

“The (B)Leading Edge: Building an Indexed File Class Using XDR_Stream”

by Jack W. Reeves
This month Jack uses the XDRStreams that he has developed in previous columns as the basis for an IndexedFile class.

“Object Interconnections: Real-time CORBA, Part 3: Thread Pools and Synchronizers”

by Douglas C. Schmidt and Steve Vinoski
Distributed real-time and embedded applications need strict control over the scheduling and execution of CPU and memory resources. This column describes Real-time CORBA’s support for these needs, focusing on its thread pools and synchronizers.
