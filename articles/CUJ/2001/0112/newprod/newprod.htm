Caldera Introduces OpenLinux Workstation 3.1

Caldera International, Inc. has released Caldera OpenLinux Workstation r3.1, a self-hosted Linux-based OS for creating applications for an open-standard environment using Java, C, and C++. Gimpel Software Releases FlexeLint for C/C++ v8.00

Gimpel Software has released FlexeLint for C/C++ v8.00, a static analyzer that can analyze a mixed suite of C and C++ programs and report on bugs, glitches, and inconsistencies. Version 8.00 debuts the Inter-function Value Tracking feature. Functions called with specific values are processed with these values used to initialize parameters. Indigo Rose Introduces Visual Patch v1.0

Crossware Adds Data Breakpoints

Crossware has added source-level data breakpoints to all of its simulators and debuggers to assist programmers of embedded systems. Breakpoints can be set on C and assembler variables to halt a program when the variables are accessed or when they match a value or range of values. A drag-and-drop interface allows these variables to be combined with each other and with execution address breakpoints to form complex breakpoint conditions.

The microcontrollers from Cygnal Integrated Products allow breakpoints to be set on up to four data memory locations or special function registers. These are used to allow hardware breakpointing on variables located in these memory areas.
