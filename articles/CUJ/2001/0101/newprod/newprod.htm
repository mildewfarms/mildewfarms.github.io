HiT Software, Inc. has announced HiT Allora for Java, which allows application developers to directly “write back” XML data to relational databases. Codemesh Debuts JunC++ion r1

Codemesh, Inc. has introduced JunC++ion r1, a patent-pending technology that allows Java and C++ programs to work together in one process. With JunC++ion, Java and C++ programmers can continue to work in whatever language best solves a problem, while relying on JunC++ion to integrate the mixed language solution. JunC++ion automatically generates C++ proxy classes that correspond to Java classes. It generates error-checking and exception-handling routines. The JunC++ion runtime library provides the low-level communication between the Java and C++ code.

JunC++ion runs Java-enabled C++ programs with all major JVMs, including Sun, IBM, and Microsoft, in both Java 1 and Java 2 environments. The JunC++ion runtime library handles differences among JVMs and Java environments.

PolySpace Presents PolySpace C Verifier

PolySpace Technologies has announced PolySpace C Verifier, a software verification and validation tool that protects against run-time errors. PolySpace C Verifier employs a technology called abstract interpretation that checks the dynamic properties of an application without actually running it. Instead of iteratively verifying software states like a case generator, PolySpace C Verifier works on an abstraction of the analyzed software.

PolySpace C Verifier detects: an attempt to read a non-initialized variable, access conflicts for unprotected shared data, referencing through null or out-of-bounds pointers, out-of-bound array access, buffer overflow, division by zero, invalid arithmetic operations such as square root of a negative number, overflow and underflow of arithmetic operations for integers and floating-point numbers, unreachable code, and illegal type conversion.

PolySpace C Verifier pricing starts at $6,000. Upper limits depend on customer needs, but price depends on number of developers, size of project, and duration. PolySpace C Verifier may also be acquired on the basis of an annual rental fee.

For more information, contact PolySpace Technologies, 877.711.7659; contact_us@polyspace.com; www.polyspace.com.
