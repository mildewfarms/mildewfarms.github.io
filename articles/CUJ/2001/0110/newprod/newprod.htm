Green Hills Offers PowerPC Compiler for the MULTI IDE

Green Hills Software, Inc. has released a PowerPC Compiler for the MULTI IDE that provides automatic vectorization support for Motorola PowerPC processors equipped with the AltiVec vector-processing engine. The PowerPC Compiler converts ordinary C code into optimized AltiVec object code.

AltiVec is a SIMD (single instruction multiple data) processing engine for Motorola PowerPC processors that enhances performance for compute-intensive applications by processing up to 16 data elements per clock cycle. The PowerPC Compiler analyzes the user’s C program, identifies code sequences that could benefit from SIMD processing, and generates object code for optimal execution on the AltiVec processor.

The PowerPC Compiler is available with the MULTI 2000 IDE. The Windows version costs $5,900. The Unix-hosted version costs $8,900.
