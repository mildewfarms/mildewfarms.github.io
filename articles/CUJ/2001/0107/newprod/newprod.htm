TurboPower Announces Memory Sleuth 2

TurboPower Software Company has shipped Memory Sleuth 2, a debugging tool for Windows programmers. Memory Sleuth 2 works by watching a Windows application while it runs and reporting errors wherever memory and other resources were allocated but never freed. It requires no changes to a program’s source code. Instead, a Memory Sleuth 2 user compiles the project with the compiler’s debugging information turned on. When Memory Sleuth 2 finds an error, it shows the programmer exactly where the error starts.

Memory Sleuth 2 costs $199 for a single-user license. Upgrades from version 1.5 cost $99. Prices include shipping and handling.

The Kernel Group Introduces AutoTrace

The Kernel Group has introduced AutoTrace, a cross-platform software analysis tool designed to work in very complex multi-threaded and distributed applications that cannot be traced with traditional debugging tools. Through its patent-pending technology, AutoTrace automatically embeds “trace points” in a company’s software. These “trace points” continually record activity information. In the event of a software failure, collecting the trace data allows companies to see exactly what transpired in the seconds leading to a software failure and resolve the problem.

AutoTrace allows the level of tracing to be adjusted at any time while a company’s program is running. For example, data points can be turned on or off to provide more or less debugging data, with no impact on the running software.

Pricing for AutoTrace starts at $5,000 per developer, with site and distribution licensing available.

Apogee Software Announces Aphelion

Apogee Software has announced Aphelion, an IDE for creating and deploying Java technology-based embedded system applications. Aphelion offers a comprehensive set of features and capabilities not typically found in Java IDEs. For example, it includes an ahead-of-time optimizer that can create compact bytecode zip files of entire deployment-ready Java applications. And to accelerate the run-time execution of bytecode methods in such zip files, Aphelion comes with a special optimizer capable of compiling user-selected performance-critical bytecode methods into highly optimized binary JNMs, each directly executable (after being invoked by Apogee’s VM through a special fast interface) on a give target processor.

The Aphelion basic configuration costs $2,990 for one development seat and includes the following tools:

  • WinNT/2000-hosted GUI with a look-and-feel of Microsoft Visual Studio
  • WinNT/2000-hosted Java/C/C++ project manager and librarian
  • Java/C/C++ syntax-driven editor
  • Java-to-bytecode optimizing compiler
  • C/C++-to-CNM compiler package (CNM: JNI-compliant “C/C++ Native Method”)
  • bytecode/native-code debugger
  • A port of Sun’s PersonalJava VM, CDC J2ME VM, or CLDC J2ME VM to user-selected target platform (CDC: Connected Device Configuration; CLDC: Connected Limited Device Configuration; J2ME: Java 2 Micro Edition)

The Aphelion complete configuration costs $4,400 for one development seat and includes the tools listed above plus the following:

  • The Bytecode Class Browser
  • JLink, the ahead-of-time bytecode class loader, linker, and space optimizer
  • The JNM speed Optimizer
  • The upgraded bytecode/native code debugger
