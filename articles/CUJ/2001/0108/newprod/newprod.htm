ParaSoft Releases Insure++ v6.0

ParaSoft Corporation has announced Insure++ v6.0, their automatic run-time error detection tool for C/C++ development. The new Chaperon mode exposes errors, such as memory corruption, memory leaks, pointer errors, and I/O errors in non-instrumented executable files without requiring any recompiling or relinking. Chaperon provides a check of all data memory references made by a process, whether in compiled code, language support routines, or shared or archived libraries. Additionally, it detects and reports reads of uninitialized memory, reads or writes that are not within the bounds of allocated blocks, and allocation errors.

Insure++ v6.0 costs $1,595 for a machine-locked, single-user license.

For more information, contact ParaSoft Corporation, 888.305.0041; fax +1.626.305.3036; info@parasoft.com; www.parasoft.com.
