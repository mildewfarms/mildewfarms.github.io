
About the "UTILS" directory on this CD-ROM
==========================================================

Some of the source code that accompanies articles on this CD-ROM is
contained in .zip and .arc archives. In the "UTILS" directory are UnZip
executables for Windows, MacOS, Linux on x86, and Solaris. We have also
included the UnZip source code for use on other platforms.

The "UTILS" directory also includes the ARC utility for MS-DOS, as well
as the ARC source code for use on other platforms.
